#  样条曲线

样条曲线工具可以用于制作环境中的街道或路径，但总体来说，可以将其用于必须贴合地貌的任意网格物体，诸如圆石墙面或整排的谷物。

<br>

## 一、创建样条曲线

### 1. 编辑样条曲线工具

在管理模式中，选择编辑样条曲线工具。

![企业微信截图_16916521533465.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916521533465.png)

### 2. 设置控制点

在当前选择的地形中，按下Ctrl+鼠标左键来设置样条曲线的首个控制点。
再次按下Ctrl_鼠标左键来创建另一个控制点，通过样条曲线段连接到首个控制点。这时，新的控制点变为选中状态，以便重复这个过程来添加更多的控制点。

![企业微信截图_1691652168821.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1691652168821.png)

### 3. 添加静态网格物体Actor

在内容浏览器中，搜寻并选择静态网格物体资源，在地形中点击样条曲线控制点；在关卡编辑器的详细信息面板的Landscape Spline（地形样条曲线）点击分段。 这样就可以确保静态网格物体被添加到所有的样条曲线段中。

![企业微信截图_16916521776238.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916521776238.png)

在 Landscape Spline Meshes（地形样条曲线网格物体）部分的Spline Meshes（样条曲线网格物体） 下，创建元素。

![企业微信截图_16916521883073.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916521883073.png)

选择静态网格物体及其贴图。

![企业微信截图_1691652208957.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1691652208957.png)

### 4. 使地貌变形为样条曲线

当地貌与样条曲线不贴合时，可以使用”使地貌变形为样条曲线“功能。

![企业微信截图_16916522189622.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916522189622.png)
