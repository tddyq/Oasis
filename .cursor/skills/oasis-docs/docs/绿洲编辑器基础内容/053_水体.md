#  水体

绿洲启元编辑器中的水体分为自定义游泳区域和水体组件两部分。

<br>

## 一、自定义游泳区域

自定义游泳区域由水体表面和水下体积两部分组成：水体表面用于显示波纹，水下体积的体积雾效果在运行后可见。
玩家角色进入该区域后，会变为游泳状态。

### 创建自定义游泳区域

![企业微信截图_16916523816596.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916523816596.png)

将自定义游泳区域拖入场景后，进行适当的缩放和调整高度，填满地形中的河流或湖泊。

![企业微信截图_16916524135008.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916524135008.png)

<br>

## 二、水体组件

水体组件用于控制地图中水上载具的行船高度。

![企业微信截图_1691652422737.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1691652422737.png)

将水体组件添加进场景后，将其高度改为与自定义游泳区域的高度一致。

![企业微信截图_1691652432608.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1691652432608.png)

地图中的水上载具就会在此高度航行。
