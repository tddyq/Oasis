#  地形Landscape

地形（Landscape）系统能够为游戏场景创建地形，比如山脉、山谷、起伏或倾斜的地面，并通过使用一系列工具修改其形状和外观。

<br>

## 一、创建地形


### 1. 地形设置

在地形工具-管理中设置地形参数，点击“创建”按钮后创建地形。

![企业微信截图_16916520596555.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916520596555.png)

- Section Size（分段尺寸）：地形用分段尺寸设置LOD和剔除。较小的尺寸使地形更积极地对分段设置LOD，但CPU的开销更高。较大的尺寸即意味着组件较少，CPU的开销较低。创建大型地形时须使用较大的分段尺寸，因为较小的分段尺寸然后放大地形比例会提高CPU的开销。
- Sections Per Component（组件分段数）：辅助地形LOD。每个分段都是地形LOD的基础单元。一个组件可能拥有2x2的分段，意味着一个组件一次可渲染四个不同的 LOD。使用较大的分段尺寸可减少CPU计算时间，从而获得额外的益处。然而，地形同时渲染数量过多的顶点时可能出现问题。使用超大的地形区域时此问题将普遍存在。这些问题在移动设备上可能尤为严重，因为绘制调用的数量将受到硬件限制。
- Number of Component（组件数）：和分段尺寸一同用于地形尺寸的设置。
- 总体分辨率：地形使用的顶点数量。
- 总组件：为地形所创建的组件总数。此数值的上限为64，因为每个组件均会产生CPU开销。超过此上限可能引起地形的性能问题。
- 填充世界：使地形尽量变大。
- 创建：使用特定设置在世界场景中创建地形。

### 2. 地形构成

![企业微信截图_16916521035351.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916521035351.png)

- Quad（四边形）：地形的最小构成单位，用深绿色显示。
- Section（分段）：用来做Quad的容器，用中绿色显示。
- Component（组件）：用来做Section的容器，用浅绿色显示。
- 地形Actor：用黄色显示。

## 二、编辑地形


完成地形创建后，可以使用不同的工具对地形进行编辑。

![企业微信截图_16916521121905.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916521121905.png)

### 1. 雕刻工具

通过雕刻工具能以当前选中笔刷和衰减的形状升高和降低地形高度图的高度：点击提升地形高度；按住Shift+点击降低地形高度。

### 2. 平滑工具

平滑工具能够柔化高度图绘制值，消除使用雕刻或腐蚀工具后有时会出现的锯齿穿帮，使地形的外观平滑流畅。

### 3. 平整工具

启用后，平整工具将把高度图的所有部分推拉至鼠标之下的水平线上。 这样便可将周围的高度图数值升高或降低到相同的数值。

### 4. 斜坡工具

使用斜坡工具可在地形上选择两个位置并在两个点之间创建一个平板斜坡，并在侧边指定衰减。

### 5. 腐蚀工具

侵蚀工具利用热力侵蚀模拟调整地形高度图的高度。它模拟土壤被自然力量从高处移动向低处的效果。 高低落差越大，产生的侵蚀效果越强。该工具还可在侵蚀上应用noise效果，呈现出自然真实的随机外貌。

### 6. 水力侵蚀工具

水力侵蚀工具利用水力侵蚀模拟（由水流形成的侵蚀）调整地形高度图的高度。Noise过滤器用于决定模拟中初始降雨的出现位置。计算出的模拟将确定初始降雨的水流，以及消融、水流迁移和蒸发。 计算结果将产生实际用于降低高度图的数值。

### 7. 噪点工具

噪点工具将应用一个噪点过滤器，在地形高度图表面生成各种变化效果。
