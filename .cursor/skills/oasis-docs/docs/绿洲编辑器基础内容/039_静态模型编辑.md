### 3. 裁剪设置

![企业微信截图_16866421982344.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866421982344.png)

- Culling Screen Size：模型裁剪尺寸设置，物体屏幕尺寸小于该数值模型将不显示。这个根据模型实际大小调整。
- Screen Size Culling Rough Distance：屏幕占比裁剪对应的裁剪距离
- Is Streamable：是否动态载入模型，蓝图类不合并模型需要勾选，mesh不勾选
