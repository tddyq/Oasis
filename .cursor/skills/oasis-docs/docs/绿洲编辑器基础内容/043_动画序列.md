#  动画序列

本章节介绍如何在绿洲启元编辑器中创建和使用过场动画序列（Sequence）。

<br>

## 基础概念

序列通常指过场动画序列。你可以将一系列元素，如Actor、组件，或是抽象的“事件”、“相机剪切” 等按时序规定他们的状态变化，组成一个“序列”。

例如，在关卡序列中规定某一相机随着时间变化，来调度镜头的轨迹，或是在Actor序列中规定某网格体组件的某材质，随着时间改变参数（如颜色）等等。绿洲启元编辑器下的序列有两种类型，第一种类型是关卡序列（LevelSequence），它用于规定某一关卡中的元素随时间的状态变化，第二种类型是Actor序列（ActorSequence），它用于规定某一蓝图下的元素随时间的状态变化。

<br>

## 关卡序列制作基础

### 1. 创建关卡序列并打开定序器（Sequencer）

在内容浏览器下新建一个`动画`-`关卡序列`的资源，双击新建的资源，打开关卡序列编辑器。

![企业微信截图_16866468958106.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866468958106.png)

### 2. 添加关卡序列轨道

- 将关卡中的Actor添加至序列轨道

点击【+轨道】按钮，在【Actor到Sequencer】子选项中找到需要的Actor，点击后即可为其添加轨道。

![企业微信截图_1686646932225.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686646932225.png)

再次点击【+轨迹】按钮，添加要编辑的Actor，如【变换】，即可创建一条用于规定Actor的变换（位置、旋转、播放）随时间变化的轨道。

![企业微信截图_16866469403255.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866469403255.png)
![企业微信截图_16866469619951.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866469619951.png)

- 随Sequence生成的Actor轨道

关卡序列不仅支持为当前关卡下的Actor添加轨道，也支持动态生成Actor并为其添加轨道，创建的Actor会在对应的关卡序列播放时动态生成到当前关卡，并有一个“⚡”标记标明它是动态生成的。

![企业微信截图_1686647629961.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686647629961.png)

- 添加抽象轨道（如相机剪切轨道）

除添加固定的Actor轨道外，关卡序列还支持一系列抽象轨道，常用的有：事件轨道、相机剪切轨道、镜头轨道、材质参数集轨道、AkAudioEvent轨道等等。

![企业微信截图_16866476571264.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866476571264.png)

### 3. 通过FBX文件导入变换轨道

关卡序列支持通过FBX文件导入变换轨道，在DCC工具（如Autodesk 3ds Max、Autodesk Maya、Blender等）制作相机变换动画并导出。可使用本章的附件【CamShow_Lseq_3.fbx】作为资产进行本小节的学习

![企业微信截图_16866476428025.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866476428025.png)

在打开的关卡序列中右击你要添加变换轨道的Actor，选择【导入】，选择第1步中制作的变换动画，取消勾选【创建摄像机】，点击【导入】按钮，一条变换轨道就会被成功创建。

`注意：在绿洲启元编辑器中制作的变换动画同样可以导出至fbx文件并再次导入。`

## Actor序列制作基础

在蓝图中也可以创建Actor序列来为蓝图中的组件设置随时间变化的属性。

### 1. 创建Actor序列并打开定序器（Sequencer）

在需要添加ActorSequence的蓝图上点击添加组件【+】按钮，在弹出的搜索框中输入【Actor Sequence】，选择【Actor Sequence】组件进行添加。同一个蓝图可以添加多个【Actor Sequence】。

![企业微信截图_1686647738486.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686647738486.png)

### 2. 编辑Actor序列

选中要编辑的【Actor Sequence】组件，在右侧的细节面板中找到并点击【在选项卡中打开】。

![企业微信截图_16866477563638.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866477563638.png)

此时定序器（Sequencer）便会自动打开并聚焦。

![企业微信截图_16866477648225.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866477648225.png)

### 3. 添加Actor序列轨道

- 将蓝图中的组件添加至Actor序列轨道

点击【+轨道▼】按钮，在【组件】子选项中找到需要的组件，点击后即可为其添加轨道。

![企业微信截图_16866477755054.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866477755054.png)

 再次点击【+轨迹】按钮，添加要编辑的组件轨道，如【变换】，即可创建一条用于规定组件的变换随时间变化的轨道。

![企业微信截图_16866477851458.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866477851458.png)
![企业微信截图_16866477956656.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866477956656.png)

- 通过带有根动画（RootMotion）的动画资产导入变换轨道

绿洲启元编辑器支持通过根动画导入变换轨道。在制作Actor序列中的骨骼网格体位移动画时，更推荐使用这种方式，而不是直接使用根动画的方式来制作角色运动轨迹，原因是将根动画转化为变换轨道后，可以为骨骼网格体对应的碰撞添加变换轨道，以此来实现碰撞与骨骼网格体同时运动的效果。

![企业微信截图_16866478045091.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866478045091.png)

导入一个具有根动画（RootMotion）的动画资产，这个资产应与即将添加至Actor序列的骨骼网格体共用同一套骨骼资产。可使用附件【Base_Run.fbx】作为资产进行本小节的学习。（动画资产导入的教程：[动画编辑](https://developer.gp.qq.com/wikieditor/#/catalog/301) ）

![企业微信截图_16866478285891.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866478285891.png)

打开已导入的动画资产，在【骨骼树】面板中找到【Root Motion】选项卡， 将其中的内容按图示配置进行修改并保存资产，即勾选【启用根运动】和【强制根锁定】，【根运动根锁定】选择【零】。

![企业微信截图_16866478365519.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866478365519.png)

打开已创建了Actor序列的蓝图，在Actor序列中添加目标骨骼网格体轨道，并为其添加【动画】和【Transform】轨道。

![企业微信截图_16866478558553.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866478558553.png)

 在【动画】轨道中为其添加第1步中导入的动画资产。

![企业微信截图_16866478739239.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866478739239.png)

添加骨骼网格体对应的胶囊体碰撞轨道，在【Transform】轨迹中，右键【从动画根导入】，同样选择第1步中导入的动画资产。

![企业微信截图_16866478837070.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866478837070.png)

此时一条变换轨道的关键帧被成功导入至当前Actor序列，当此Actor序列播放时，该骨骼网格体的胶囊体碰撞便会与角色同步运动。

![企业微信截图_16866480743078.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866480743078.png)
