#  编辑器资源库

绿洲启元编辑器内置了大量和平精英模型，动画，特效等美术资源，开发者可以直接使用这些资源进行玩法创作。

<br>

## 资源分类

### 1. UI

- UI图片：内置的UI图集资源；
- UI资源：包含贴图资源，图集资源，特效材质资源；
- UI预设：完整可用的UMG资源，包含加载类、商店类、战斗类、指引类、提示类、玩法引导类

### 2. 地图

- 关卡：包含一些和平精英子关卡相关资源；
- 出生点：包含关卡出生点的配置蓝图；
- 喷涂：包含一些和平精英的拾取喷涂资源；
- 建筑：包含了大量和平精英的建筑资源；
- 植物：包含了大量和平精英的植被资源；
- 物件：包含了大量和平精英的物件资源；
- 石头：包含了大量和平精英的石头资源；
- 贴图：包含一些地形贴图材质和贴图资源；
- 道路：包含一些道路资源模型

### 3. 怪物

- 包含一些怪物资源

### 4. 游戏模式

- 模式：包含模式配置蓝图；
- 模式物件：包含空投配置，信号圈等模式物件；
- 经典模式：包含一些游戏阶段状态的配置内容；
- 行为树
- 表格：包含空投数据、海岛道具表

### 4. 物品

- 投掷物：和平精英投掷物资源；
- 枪械：和平精英枪械资源；
- 枪械配件：和平精英枪械配件资源；
- 药品：和平精英药品资源；
- 表情：和平精英表情资源；
- 装备：和平精英装备资源；
- 近战武器：和平精英近战武器资源；
- 配件： 和平精英配件资源

### 5. 皮肤

- 包含拾取文件、图标、皮肤美术资源分类

### 6. 美术资源

- 僵尸：包含多种类型的僵尸资源；
- 动作：包含多种角色和僵尸的动画资源；
- 图标：包含皮肤、物品等图标资源；
- 材质：包含几大类可继承的材质实例资源；
- 模型：包含多种类型的模型资源；
- 特效：包含多种类型的特效资源；
- 角色：包含完整的其他类角色美术资源；
- 骨架：包含多种类型的骨骼资源

### 7. 行为树

- 包含怪物行为树资源

### 8. 表格

- 包含和平精英物品表资源

### 9. 角色

- 技能：内置的UI图集资源

### 11. 载具

- 包含主角PlayerPawn,PlayerState,角色技能，角色Buff等配置文件

### 12. 音效

- 包含多种类型的大量音效资源；
