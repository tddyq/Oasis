#  特效编辑

基础的特效内容操作规范

## 特效资源设置

特效内容编辑需要用到贴图和材质的相关内容，特效应用到的贴图内容需要根据规范来设置，特效材质可使用编辑器默认的四种材质进行编辑

### 特效贴图

---

![企业微信截图_16866464432417.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866464432417.png)
CompressionQuality：默认使用TCQ_Default<br>
Compression Quality：TCQ_Highest<br>
Mip Gen Settings:TMGS_NoMipmaps<br>
Texture Group:TEXTUREGROUP_UI

### 特效材质介绍

![企业微信截图_1686646459428.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686646459428.png)
特效材质1：<br>
所有像素值叠加不暗化，黑色渲染是透明的。例如火焰、灯光类效果。<br>
特效材质2：<br>
半透明，黑色透明，白色完全不透明。例如烟。<br>
特效材质3:<br>
特效材质1的基础上增加材质纹理的动态效果。<br>
特效材质4:<br>
特效材质2的基础上增加材质纹理的动态效果。
