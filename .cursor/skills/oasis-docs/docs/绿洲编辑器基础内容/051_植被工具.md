#  植被工具

植被工具是一种用于在其他几何体表面上渲染静态网格体或植被Actor的模式，可用于表现地面上的覆盖物效果。

<br>

## 一、创建植被

### 1. 植被编辑模式

点击选择植被编辑模式

![企业微信截图_16916523329381.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916523329381.png)

### 2. 添加植被类型

点击添加植被类型

![企业微信截图_16916523391837.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916523391837.png)

### 3. 放置植被

勾选想要放置的植被类型后，在场景中点击鼠标左键放置。

![企业微信截图_16916523518799.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916523518799.png)

**注意：如果需要放置单株的植物，则需要需要勾选“Single Instance Mode"**

![企业微信截图_16916523608286.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16916523608286.png)
