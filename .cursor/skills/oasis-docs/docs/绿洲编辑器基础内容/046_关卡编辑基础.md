#  关卡编辑基础

## 关卡简介

- **关卡：** 绿洲启元编辑器中，所有玩家看到的对象和交互的对象，我们都可以保存在一个“世界“”里，这个“世界”我们称之为关卡，关卡中可以保存编辑器中大部分的资源类型。
- **永久性关卡** ：工程的主关卡，每个项目始终都会有一个永久性关卡。
- **子关卡** ：也是独立的关卡文件，可通过不同方式与主关卡建立连接管辖。

<br>

## 关卡初识

管理多个关卡：可通过【关卡】窗口进行管理，在菜单栏中可找到。

### 1. 创建子关卡

- 新建：单击 【关卡】下拉菜单，然后选择 【新建关卡】，可新建一个子关卡。
- 添加：单击 【关卡】 下拉菜单，然后选择 【添加现有关卡】，可添加一个现有关卡为子关卡。
- 拆分：先在场景大纲中选中要拆分的物件，然后单击 【关卡】 下拉菜单，然后选择 【新建带有选中物体的关卡】，可为选中的物体们创建一个新的子关卡。

### 2. 编辑保存子关卡

双击可选中要编辑的子关卡，才可对其进行编辑，蓝色为当前选中关卡。下图蓝色为选中态，有小铅笔表示内容有修改，需要保存。

### 3. 关卡细节查看

关卡信息查看面板：可以设置子关卡的旋转 位置 和显示颜色等

### 4. LevelBounds

 LevelBounds代表关卡的边界。会根据当前关卡中所有物体的的位置，自动生成该关卡的范围边界，用来计算该关卡大小，不可人为修改LevelBounds大小，但是改变关卡中物体的位置。
 
![企业微信截图_168664847056.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_168664847056.png)

### 5. 子关卡编辑器内的显示和隐藏

先选中关卡，选中的关卡为蓝色。双击选中要编辑的子关卡，所有的新增编辑都只会对当前选中的子关卡生效（所有新加入场景的资源，会被保存到当前选中的子关卡中）。被load出来的关卡为白色，在编辑器内可见，未被load出来的关卡为灰色，在编辑器内不可见。
