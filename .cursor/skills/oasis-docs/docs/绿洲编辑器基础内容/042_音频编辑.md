#  音频编辑

音频的导入和编辑操作规范

## 导入格式

mp3<br>
wav<br>
aac

## 导入选项

![企业微信截图_16866467866486.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866467866486.png)
编辑器提供了2D、3D、Weapon、Vehicle、Character 5类音频模板。<br>
2D,3D音效可基本覆盖玩法逻辑内容相关需求；武器，枪械，角色音效需要结合和平精英内相关逻辑做适配。<br>
导入音频默认使用2D模板，勾选其他类型可切换模板类型导入

## 内容编辑

### 基础参数设置

![企业微信截图_16866468087511.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866468087511.png)
Volume：音量调整<br>
Pitch：音高调整<br>
Max Attenuation Radius：音效衰减距离<br>
Is Infinite：音频是否循环播放

### 事件音频接入

![企业微信截图_16866468161352.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866468161352.png)
Block Sounds：Target音频播放后会打断该音效<br>
Volume Controls：Target音频播放后会将该音频音量调整到某个值
