# 道路生成器

在绿洲启元编辑器中，开发者可以使用道路生成器，快速编辑并生成道路。它由样条线组件（Spline）和程序化生成（PGC）相关的规则组成，通过调整曲线和改变配置规则，可以生成不同效果的道路。

![ScreenShot_2025-11-11_175549_893.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/v3cCmScreenShot_2025-11-11_175549_893.png)

<br>

## 道路生成工具

在模式编辑中选择【PCG工具箱】进入PCG编辑模式，然后选择需要生成的道路形式，目前支持道路和十字路口两种类型。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/NhCVDimage.png)

在场景中按下Ctrl+左键生成道路，生成道路后可以按End键盘使当前道路贴合地面。

![生成道路 (2).gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/FEq6U%E7%94%9F%E6%88%90%E9%81%93%E8%B7%AF%20(2).gif)

生成的道路会显示相关曲线信息，包括曲线节点序号、相邻节点间的距离、曲线的总长度。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/WWnEaimage.png)

如需隐藏这些信息或调整显示效果，可进入【工程设置】进行自定义配置，配置内容及说明如下：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/oQD4limage.png)

|属性名称|说明|
|:-:|-|
|显示PCG Hud|勾选后将显示道路的曲线信息|
|节点颜色|曲线节点序号的显示颜色|
|节点显示缩放|曲线节点序号的显示大小|
|距离颜色|相邻节点间距离数值的显示颜色|
|距离显示偏移|距离数值相对于道路的垂直显示位置，值越大则位置越高|
|距离显示缩放|相邻节点间距离数值的显示大小|
|总距离颜色|曲线总长度数值的显示颜色|
|总距离显示偏移|总长度数值相对于道路的垂直显示位置，值越大则位置越高|
|总距离显示缩放|曲线总长度数值的显示大小|

<br>

## 路线编辑

选择道路的任一节点，按住Alt键并拖动，可快速复制生成新的路径节点，延伸路径。

![延伸路径.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/4JLwd%E5%BB%B6%E4%BC%B8%E8%B7%AF%E5%BE%84.gif)

移动各个节点，可调整路径的整体走向与大致轮廓。

![走向与轮廓.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/MDS2a%E8%B5%B0%E5%90%91%E4%B8%8E%E8%BD%AE%E5%BB%93.gif)

通过缩放节点，可精确调节该节点路径的弯曲效果或放大效果。

![缩放节点.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/LrDxB%E7%BC%A9%E6%94%BE%E8%8A%82%E7%82%B9.gif)

## 道路吸附

道路生支持相互吸附功能，当两条道路靠近时，系统会根据曲线索引值自动将它们首尾相连，实现无缝衔接。

![吸附.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ZltEV%E5%90%B8%E9%99%84.gif)

通过多条道路的连续吸附连接，可构建环形道路等复杂路径。

![image.40.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/p52Kiimage.40.png)

选中道路后，在【细节】面板中找到【Snapping Setting】属性组，可调整以下吸附参数：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/z8Q28image.png)

|属性名称|说明|
|:-:|-|
|Radius|吸附检测的半径大小，单位：厘米|
|Invert Start|反转起点|
|Invert End|反转终点|

当制作首尾相连的道路时，如果两条曲线的角度偏差较大，可能会出现两个连接点朝向相同的情况，此时可以通过调整"Invert Start"或"Invert End"来修正道路的朝向，确保连接处的平滑过渡。

![invertEnd.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/knEoxinvertEnd.gif)

<br>

## 道路模型规范

开发者可以根据自身需求制作路面模型，导入的模型中，Pivot需在最左侧，道路延申方向为X轴正轴。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/v2sdwimage.png)

细分需保持合适的程度，细分太多会影响性能，细分太少会影响生成道路的效果

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ZNIBWimage.png)

<br>

## 道路材质规范

编辑道路材质时，必须基于“地砖材质”创建材质实例进行贴图和参数的设置，否则生成路面的效果可能出现异常。

> 材质路径：和平精英/美术资源/材质/默认
