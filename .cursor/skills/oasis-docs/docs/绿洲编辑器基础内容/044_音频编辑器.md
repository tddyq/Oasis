# 音频编辑器

音频编辑器是编辑器中用于音效资源管理的工具，支持音频资源的导入、剪辑、属性配置等操作。

<br>

## 音频编辑器界面

在编辑器主界面的工具栏，点击进入【资产编辑器】，然后选择【音频】，将打开音频编辑器的操作界面。

![音频管理器界面.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/l9r5u%E9%9F%B3%E9%A2%91%E7%AE%A1%E7%90%86%E5%99%A8%E7%95%8C%E9%9D%A2.drawio.png)

音频编辑器的界面图以及各部分的说明如下：

![音频管理器-结构 的副本.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/rvGKy%E9%9F%B3%E9%A2%91%E7%AE%A1%E7%90%86%E5%99%A8-%E7%BB%93%E6%9E%84%20%E7%9A%84%E5%89%AF%E6%9C%AC.drawio.png)
|序号|页面名称|功能说明|
|:-:|:-:|-|
|A|音频浏览区|展示已创建的音频资源，可查看、选择音频|
|B|音频预览区|可视化显示音频波形图，支持音频裁剪、定位、渐入渐出操作|
|C|属性设置区|配置音频参数，包括基础设置、裁剪设置和高级设置|
|D|音频操作区|提供音频文件的基础操作功能，包括导入、播放、保存和停止|

<br>

## 音频功能

### 创建音频

音频创建支持外部文件导入与模板快速创建两种方式，开发者可根据项目需求灵活选择。

#### 外部导入音频

首先点击“导入”按钮。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ksClPimage.png)

然后在文件夹窗口单选需要导入的音频文件，点击“打开”，目前仅支持全英文的文件路径，支持的音频格式包括：.mp3、.wav、.aac。

![ScreenShot_2025-11-03_154158_464.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/85BDWScreenShot_2025-11-03_154158_464.png)

最后在窗口“Audio导入选项”的【Custom】属性组中选择音效类型，点击“导入”，完成外部导入音频。

![ScreenShot_2025-11-03_154619_153.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ongZkScreenShot_2025-11-03_154619_153.png)

音效类型共有三种，它们的特点如下：

|类型|说明|
|:-:|-|
|2D|音频音量不会随听者与声源距离而变化，同一时刻只能播放一个2D音效|
|3D|空间化音效，音频音量随听者与声源距离增大而衰减|
|BGM|音频音量不会随听者与声源距离而变化，同一时刻能播放多个BGM音效|

---

#### 使用音频模板创建

在【音频浏览区】的【Template】分类中，单击选择音频模板，然后点击“选择模板创建”，最后在弹出的【输入名称】窗口中输入音频名称，点击“确定”即可完成创建。

![音频管理器-创建.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/7Vr9X%E9%9F%B3%E9%A2%91%E7%AE%A1%E7%90%86%E5%99%A8-%E5%88%9B%E5%BB%BA.drawio.png)

目前通过模板创建的音频仅支持基础设置编辑，且不会出现音频预览区。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/BiecPimage.png)

---

### 音频播放/停止

选中目标音频后，点击“播放”按钮，音频将从裁剪设置中定义的开始时间进行播放；点击“停止”按钮即可立即终止播放。

![音频管理器-播放与暂停.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/klKrg%E9%9F%B3%E9%A2%91%E7%AE%A1%E7%90%86%E5%99%A8-%E6%92%AD%E6%94%BE%E4%B8%8E%E6%9A%82%E5%81%9C.drawio.png)

### 音频编辑

在音频编辑器的【属性设置区】中，可以对选中的音频进行详细编辑。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/WIrpWimage.png)

基础设置的内容及说明如下：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/0zl2rimage.png)

|属性名称|说明|
|:-:|-|
|音量|控制音频的整体响度，默认值为0.0，单位：dB|
|音高|控制音频的音调高低，默认值为0.0，单位：Hz|
|循环|设定音频播放结束时是否自动重新开始播放|
|衰减范围|设定音效在场景中可被听到的最大距离，【音效类型】选择“3D”时有效，单位：厘米|
|音效类型|定义音频的表现模式，可选：2D、3D、BGM|
|中断事件|当前音频播放时，会中断指定音频事件的播放，其包含以下子属性：<br>◦ 目标事件：指定要被中断的音频事件<br>◦ 是否全局：若勾选，则中断场景中所有正在播放的该目标事件；若不勾选，则仅中断当前音频所在对象中的目标事件|
|改变音量|当前音频播放时，会修改指定音频事件的音量，其包含以下子属性：<br>◦ 目标事件：音量会被修改的音频事件<br>◦ 期望值：目标事件被修改后的音量值，单位：Hz<br>◦ 是否全局：若勾选，则修改场景中所有正在播放的该目标事件的音量；若不勾选，则仅改变当前音频所在对象中的目标事件音量|

裁剪设置的内容及说明如下：

![ScreenShot_2025-11-03_173929_782.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/XnpEWScreenShot_2025-11-03_173929_782.png)

|属性名称|说明|
|:-:|-|
|开始（毫秒）|音频开始播放的时间点|
|结束（毫秒）|音频结束播放的时间点|

除开通过裁剪设置进行音频裁剪，也可以通过拖动音波图下方的蓝色滑块，改变当前音频的开始和结束时间，实现音频裁剪。

![剪音频.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/kdMKg%E5%89%AA%E9%9F%B3%E9%A2%91.gif)

高级设置的内容及说明如下：

|属性名称|说明|
|:-:|-|
|渐入时间|音频从开始播放到完全渐入的持续时间，单位：毫秒|
|渐入曲线|音频渐入方式，可选：Linear（线性）和Sine（正弦曲线）|
|渐出时间|音频从开始渐出到完全结束的持续时间，单位：毫秒|
|渐出曲线|音频渐出方式，可选：Linear（线性）和Sine（正弦曲线）|

除开通过高级设置改变音频的渐入渐出时机，也可以通过拖动音波图上方的蓝色滑块，改变当前音频的渐入渐出时机。

![渐入渐出.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/QWnmS%E6%B8%90%E5%85%A5%E6%B8%90%E5%87%BA.gif)

完成编辑后，需点击【音频操作区】的“保存”按钮，使所有修改生效。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/29Jkuimage.png)
