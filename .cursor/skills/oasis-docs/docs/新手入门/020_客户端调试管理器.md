# 客户端调试管理器

[编辑器Debug调试](https://developer.gp.qq.com/wikieditor/?timeStamp=1725589224096#/catalog/307) 可以启动指定玩家数量的客户端，能够满足常规的调试需要，但该方式仅能响应调试的启动/取消而创建/关闭客户端进程，客户端调试管理器则提供了更加便捷和精细化管理客户端的能力，开发者可以使用该工具进行更深度的调试测试。

<br>

## 客户端管理

### 创建客户端

点击编辑器菜单栏 ``打开UGC客户端管理器``，弹出管理器窗口，展开【创建客户端】按钮的下拉选项，输入 **ClientId（客户端id）** 和 **TeamId（队伍id）** 并点击【创建客户端】按钮，编辑器将会启动客户端，且当前客户端的进程id、客户端id、队伍id、玩家PlayerKey、玩家PlayerName等信息将展示在管理器的列表中。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/JNcNdimage.png)
![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/RXT8Oimage.png)

通过 [Debug调试](https://developer.gp.qq.com/wikieditor/?timeStamp=1725589224096#/catalog/307) 启动的客户端进程信息也会体现在管理器中，开发者可以利用客户端标题中的玩家名称或者进程号标识来区分不同的客户端。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/a7Nezimage.png)

---

### 重启客户端

选择要重启的客户端进程，点击【重启客户端】按钮，客户端将被关闭、重新启动，并再次连入服务器，玩家的最新可复制属性数据将被DS自动同步下来。

开发者可以通过此功能特性模拟玩家冷启动型断线重连的场景，例如杀进程重连、客户端闪退等，以验证重连后各项数据和功能是否恢复正常。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/nkVi8image.png)

---

### 终止客户端

点击管理器【终止客户端】按钮，将会关闭选中的客户端进程，该功能可以模拟玩家退出游戏的场景，用以验证DS逻辑和其他客户端表现是否正常。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/z4OmQimage.png)

## 弱网模拟

高质量的玩法需要考虑玩家的不同游戏环境，网络较差的情况下产生的数据丢包、高延迟等问题会对玩法的逻辑、表现甚至个人数据产生影响，对此需要针对性做一些 [性能优化](https://developer.gp.qq.com/wikieditor/?timeStamp=1725589224096#/catalog/348) 和异常保护机制以保障玩法的稳定运行和玩家的良好体验，客户端管理器提供了参数化的配置方式帮助开发者进行各类网络环境的模拟。

选择需要进行弱网模拟的客户端进程，点击【网络控制选项】按钮，在弹出的选项框中有六种参数可供开发者自定义填写：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/2lE1Aimage.png)

|参数名称|参数解释|测试场景|
|:-:|-|-|
|Loss|丢包率，取值[0,100]，单位为百分比|模拟网络不稳定（如Wifi信号弱）|
|Dup|重复发包，取值[0,100]，单位为百分比|模拟网络不稳定（如Wifi信号弱）|
|Order|乱序发包，1为开启，0为关闭|模拟真实设备连接DS的网络环境|
|Lag|网络延迟，单位为ms|模拟不同网络环境（Wifi约70ms，蜂窝网络约100ms）|
|LagVariance|随机网络延迟，单位为ms|模拟网络不稳定（如Wifi信号弱）|

不同弱网场景下的参数值建议：

|场景|Loss|Dup|Order|Lag|LagVariance|
|:-:|:-:|:-:|:-:|:-:|:-:|
|稳定有线网络|0|0|1|0|0|
|稳定4G/5G网络|0|0|0|100|0|
|稳定Wifi网络|0|0|0|70|0|
|不稳定网络（信号弱）|5|0|0|200|0|
|不稳定网络（地铁上5G）|10|0|0|300|0|
|不稳定网络（跨服高延迟）|5|0|0|300|0|
