# 玩法提审与更新

当玩法开发完成之后，开发者便可以将玩法提交到和平精英绿洲启元模式中了，但是在正式面向玩家之前，官方会对提交的玩法进行审核；此外，对玩法持续地进行内容更新和迭代优化，有助于保持对玩家的吸引力，而更新的内容也同样需要被审核。下面将对初次创建项目及玩法提审，与玩法更新时提审的操作步骤分别进行介绍。

<br>

## 创建项目及玩法提审
首先通过绿洲启元编辑器上传玩法工程（具体操作可参考 [上传工程与真机测试](https://developer.gp.qq.com/wikieditor/#/catalog/372)），上传成功后点击“点击这里，申请一个新的项目”选项，会打开绿洲启元管理平台，也可以直接通过链接 [https://gpeditor.qq.com/auth](https://gpeditor.qq.com/auth) 前往（这里记录一下jobid：11401，之后在配置项目信息时会用到），注意：使用`发布上传`功能上传的版本才可以在绿洲启元管理平台进行项目绑定。

## 玩法更新提审
当玩法已经通过审核后还需要对版本内容进行更新时，开发者也需要对更新的内容进行提交审核。在绿洲启元编辑器中，上传新版本的工程后，记住job id，然后前往绿洲启元管理平台，选择要更新的项目点击“查看详情”，如图2-1。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/DF50bimage.png)
页面整体与提审类似，但是在填写的内容上有所区别，需要向玩家传递更新的内容信息以提升用户的感知，这里提供一些填写技巧供参考：
1. 如果更新内容包含新增地图介绍或者新增关卡介绍，可以在“项目简介”处加上新地图或新关卡的简要描述信息；
2. “版本更新说明”需要详细填写更新的内容，例如新增XXX地图/关卡、优化了XXX体验等等，页面上也提供了“一键填入模板”功能，帮助开发者快速填写；
3. 详情页头图可放新地图或者新关卡图片，重点突出更新内容，可以适当加上文案辅以说明具体内容，例如“新上线XXX地图”等；
