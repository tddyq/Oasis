#  绿洲启元编辑器基础术语

绿洲启元编辑器基于UE4游戏引擎的基础上开发，部分概念和基础术语与UE4基本相同，可阅读UE4官方介绍文档
[https://docs.unrealengine.com/4.27/zh-CN/Basics/UnrealEngineTerminology/](https://docs.unrealengine.com/4.27/zh-CN/Basics/UnrealEngineTerminology/)

这里仅列出UE4术语介绍没有涉及的部分内容。

### 调试/Debug/PIE

绿洲启元编辑器上的调试按钮可以模拟游戏运行调试环境，以上说的名词均是指调试。

### 战斗服务器/DS

和平精英服务器区分为战斗服务器（DS）和大厅服务器，开发者开发玩法的时候主要使用的是战斗服务器，DS也是战斗服务器的代称。

### 日志/LOG

在调试结束后，或者Windows客户端对局结束后，会在对应的目录产生日志（LOG）文件，里面会带有开发者加入的print信息，以及部分关键报错内容，可以用于辅助定位开发中遇到的问题。

### Lua脚本

在绿洲启元编辑器中使用Lua脚本替代了原本UE4的蓝图脚本编辑，点击蓝图上方的Lua按钮会自动在对应路径生成Lua脚本，在Lua脚本中可以开发所需逻辑。
