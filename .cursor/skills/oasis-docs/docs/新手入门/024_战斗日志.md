# 战斗日志

为方便开发者在调试中查看战斗相关的日志信息，编辑器提供了全新的战斗日志功能，可实时显示关键事件的打点信息。

## 战斗日志界面

进入调试后，点击设置按钮左侧的第一个功能键，即可打开战斗日志窗口。

![战斗日志.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/8IqXB%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97.drawio.png)
![ScreenShot_2025-10-28_161002_769.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/9cNu5ScreenShot_2025-10-28_161002_769.png)

点击战斗日志窗口右上角的"×"按钮，即可关闭日志窗口。

![战斗日志-关闭战斗日志.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/bx5wn%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97-%E5%85%B3%E9%97%AD%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97.drawio.png)

点击日志区域底部的"点击跳转到最新"按钮，可快速跳转至最新生成的日志条目。

![战斗日志-点击跳转到最新.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/erB4a%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97-%E7%82%B9%E5%87%BB%E8%B7%B3%E8%BD%AC%E5%88%B0%E6%9C%80%E6%96%B0.drawio.png)

## 日志筛选

### 条件筛选

点击战斗日志窗口左上角的“筛选”按钮，可展开或收起条件筛选面板。

![战斗日志-展开筛选.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/dczuJ%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97-%E5%B1%95%E5%BC%80%E7%AD%9B%E9%80%89.drawio.png)

在筛选面板中，可从多个维度精确过滤日志，选择筛选条件后，点击右下角的“筛选”按钮即可显示符合条件的战斗日志。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/3TlAtimage.png)

点击“取消全选”可一键清除所有已选条件，再次点击可恢复全选状态。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/OBMmVimage.png)

---

### 时间筛选

点击战斗日志窗口上方的时间显示区域，即可展开或收起时间筛选面板。

![战斗日志-展开时间筛选.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/76Udy%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97-%E5%B1%95%E5%BC%80%E6%97%B6%E9%97%B4%E7%AD%9B%E9%80%89.drawio.png)

展开时间筛选面板后，首先点击“开始时间”，通过滚动选择器设定查询的起始时间点，然后点击“结束时间”，设定查询的截止时间点，最后点击右下角的“筛选”，日志窗口将立即显示指定时间段内的战斗日志。

![ScreenShot_2025-10-28_153258_232.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/8IrG1ScreenShot_2025-10-28_153258_232.png)

点击"恢复默认"按钮，系统将自动设定以当前时间为基准，前后各延伸3分钟的时间范围（共6分钟），并立即执行筛选。

![战斗日志-恢复默认.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/I0tPE%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97-%E6%81%A2%E5%A4%8D%E9%BB%98%E8%AE%A4.drawio.png)

## 日志搜索

在战斗日志窗口右上角的搜索框中输入需要查找的关键词，点击左边的搜索按钮后，系统将自动筛选包含该关键词的所有日志。

![战斗日志-搜索关键词.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/s7waB%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97-%E6%90%9C%E7%B4%A2%E5%85%B3%E9%94%AE%E8%AF%8D.drawio.png)

点击搜索框右侧的"×"按钮，可立即清除当前关键词，恢复显示全部日志。

![战斗日志-清空.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/YhvVC%E6%88%98%E6%96%97%E6%97%A5%E5%BF%97-%E6%B8%85%E7%A9%BA.drawio.png)

## 打印战斗日志

通过 ``ugcprint(content)`` 打印的日志也会同步显示在战斗日志中。显示格式为：时间戳+打印内容，如下图所示：

![ScreenShot_2025-10-29_153330_429.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/eOiyyScreenShot_2025-10-29_153330_429.png)|

如果用 ``ugcprint(content, category)`` 的形式进行日志打印，则category的目录会出现在战斗日志的自定义页签里，也可以用该分类进行过滤。

```lua
ugcprint("click trigger", "SkillSlot")
```

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/mRCdUimage.png)

<br>

## 日志输出环境

战斗日志（包括通过 ugcprint打印的自定义日志）仅在外研线和 PIE 环境中输出，现网环境不会输出。
