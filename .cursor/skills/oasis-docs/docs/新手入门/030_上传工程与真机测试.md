# 上传工程与真机测试

在玩法开发过程中，开发者需要在本地环境和真机环境下对玩法进行测试，以验证玩法的质量。其中本地环境可以通过 [启动调试游戏](https://developer.gp.qq.com/wikieditor/#/catalog/307) 基于PIE的运行环境进行测试；而对于真机环境，绿洲启元编辑器提供了一套便捷的调试工具来予以支持，下面将介绍如何在真机上测试玩法。

<br>

## 设置白名单

在进行真机测试前，需要设置测试玩法可见人员白名单。打开测试上传，在上传面板中，可以填写“白名单”。点击“+”按钮可以为白名单数组添加元素，白名单最大个数为30。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/It8vlimage.png)

开发者需要将测试人员的和平精英账号UID填写到白名单中，UID可以通过登录绿洲启元编辑器提供的安卓游戏客户端获取，客户端安装包的目录路径可参考：``…\WeGameApps\rail_apps\和平精英绿洲启元编辑器(2001776)\安卓``。

![image_1693985856.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/BVHojimage.png)

安装并登录客户端后，点击头像进入用户详细信息界面。

![image_1693985857.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/rv3egimage.png)

在用户详细信息界面可以看到编号，该编号即为玩家的UID，点击右侧的复制按钮复制UID。

![image_1693985858.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/IxyQlimage.png)

将UID复制到白名单中，设置白名单完成。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/OoOAtimage.png)

<br>

## 上传工程

设置完白名单后，就可以将项目工程进行上传

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/clwUlimage.png)

窗口中会显示发布平台选项，包括Windows、Android和IOS平台

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/VQHXGimage.png)

上传工程需要一定时间，上传完毕后，上传窗口中会有“上传成功”的提示。

![image_1693985862.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/6RYPsimage.png)

工程上传成功后，便可在客户端中测试玩法，可以点击“打开安卓包目录”快捷跳转到客户端安装包的目录。

<br>

## 玩法测试

启动并登录客户端后，在大厅选择绿洲启元模式，找到刚上传的工程（根据项目名称），下载工程后即可选择玩法进行测试。
<span style="color:red">**绿洲启元模式需要在账号等级达到5级后解锁，如果等级不够可以使用大厅界面右上角的GM功能来给账号增加经验**</span>。

![image_1693985863.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/9Y0CKimage.png)
![image_1693985864.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ZnGPqimage.png)

玩法的大小与实际工程中包含的资源有关，开发者在上传工程前，应对工程资源进行优化并删掉无用资源，以减小包体大小。另外，<span style="color:red">**要保证上传工程所用的编辑器版本和客户端安装包的版本相匹配（至少保证前2位版本号一致）**</span>，版本信息可以在各自登录界面看到。

![image_1693985865.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/WT89Vimage.png)
![image_1693985866.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/OMxEsimage.png)

如果希望模拟玩法在和平精英绿洲启元大厅的上架状态，则需要使用`发布上传`，并在绿洲启元管理平台的绑定项目。注意`测试上传`的的版本无法在绿洲启元管理平台绑定项目，仅`发布上传`的版本可以进行绑定

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/NQOkRimage.png)
![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/MdYXPimage.png)

登录 [绿洲启元管理平台](https://gpeditor.qq.com/user#myprojects) 并进入「玩法项目列表」页面，找到项目工程相关联的对应项目名称（如果尚未创建过项目，可以参考 [玩法提审与更新](https://developer.gp.qq.com/wikieditor/#/catalog/378) 来执行相关操作），点击“查看详情”按钮进入版本列表窗口，再次点击上方的“测试账号管理”按钮，会弹出测试账号白名单填写界面。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/PJ0Bqimage.png)
![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Yo3Rdimage.png)

将UID填入用户信息输入框（账号数量可根据实际情况自行添加，最多可支持30个测试账号），点击“开启测试”按钮二次确认白名单后开启测试即可进入客户端进行验证。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/oc0fIimage.png)
![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/mT5g7image.png)

如果开启测试时该玩法未设置白名单账号，此时需要添加白名单后才能开启测试

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/v48y0image.png)

开启测试后将在玩法列表页和玩法详情页展示配置的内容，这些内容均来自于创建项目时填写及上传的素材。
- 玩法列表页包含项目名称、玩法封面图、玩法图标、玩法分类及玩法标签等信息。

![image_1693985869.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/j54tZimage.png)

- 玩法详情页包含项目名称、项目简介、详情页各位置图或视频、开发者及玩法标签等信息。

![image_1693985870.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/pctWIimage.png)

验证完成后在版本列表窗口点击“关闭测试”按钮，即可关闭玩法相关配置内容的展示（不影响玩法的功能测试）。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/k1x5ximage.png)
