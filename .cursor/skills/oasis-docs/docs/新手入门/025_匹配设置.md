# 匹配设置

匹配是指将和平精英大厅中不同的组队/个人凑成战斗所需要的队伍数和人数的过程，在绿洲启元编辑器中，匹配无须开发者写代码，只需要简单的配置即可实现匹配。

#### 1.1 队伍设置

- 【队伍数量】：指需要大厅丢到战斗的队伍数
- 【小队玩家数量】：指每个队伍的玩家数
- 【注意】：队伍数量是指大厅丢给战斗的队伍数，并不意味着一定等于战斗内的实际分队情况。例如：《小红帽与大灰狼》玩法中配置的队伍数是1，小队玩家数量是9，那么大厅在匹配时会将9名玩家丢给战斗，之后，《小红帽与大灰狼》工程中再对这9人重新分队，分成6小红帽vs3大灰狼

#### 1.2 匹配方法

- 【匹配方法】：
  1. PeaceClassic: 用和平经典段位分匹配，适用于类经典模式或者钢枪型玩法
  2. Random: 完全随机匹配，适用于纯娱乐型玩法
  3. UserDefined：用户自定义分数匹配，适用于玩法会有水平差异，但又不适合用经典模式的段位分（例如小红帽与大灰狼玩法）
- 【初始分数】：仅当选择UserDefined时生效，指玩家初始的匹配分，定死2000
- 【最低分数】：仅当选择UserDefined时生效，指玩家最低的匹配分，定死0
- 【最高分数】：仅当选择UserDefined时生效，指玩家最高的匹配分，定死4000
- 【匹配小段分数】：每100分为一个匹配分段（类似和平，青铜5到青铜4为100分）
- 【匹配扩段时间】：默认10，指10秒内只在同分数段内的人匹配，超过10秒扩到+-1段，再超过10秒扩到+-2段，以此类推
- 【匹配扩段数量】：默认10，指最多扩到+-10段
- 【组队匹配分计算】：
  1. AverageScore:指组队的人按组队的平均分计算匹配分
  2. HighestScore:指组队的人按组队中的最高分计算匹配分
