#  使用Lua编写玩法功能

绿洲启元编辑器使用Lua作为主要的脚本语言，在创建的可编程蓝图中上方有一个Lua按钮，点击后会在工程Script文件夹对应的路径下创建一个同名的Lua脚本，使用Lua脚本可以监听对应蓝图事件，调用蓝图函数。

具体的Lua脚本开发规则： [绿洲启元Lua脚本开发指南](https://developer.gp.qq.com/wikieditor/#/catalog/346) 。
