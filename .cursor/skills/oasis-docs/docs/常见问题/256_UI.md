# UI

## Q1：为什么在移动、滑动或操作其他控件时，无法正常点击UI按钮

这通常是因为按钮或其他控件的 IsFocusable 属性被启用，导致它们在接收输入焦点时与其他操作冲突。关闭该属性可以避免输入被拦截，从而恢复按钮响应。

<br>

## Q2：如何隐藏【退出观战】控件

适当的时机下在PlayerController的客户端执行 ``self:CastUIMsg("UIMsg_HideQuitWatch", "")``。

<br>

## Q3：如何为玩法添加和平精英主模式的击杀播报UI

在GameMode蓝图中配置，找到GMDataManager - Global Settings - UI模块预置，将KillInfoModule填入。如果有多个模块需要填入，可用|符分割。

## Q4：如何将图片资源加载并显示在Image控件中

相关代码如下：

``` lua
-- 加载纹理资源
local Texture = UGCObjectUtility.LoadObject(imagePath)
-- 将纹理资源设置到Image控件的Brush属性中
self.Image:SetBrushFromTexture(Texture, true)
```

<br>

## Q5：为物件对象蓝图添加了Widget组件，为什么运行时更新绑定的UI控件变量值，但是控件效果却无法正常响应变化

使用 ``Widget:RequestRedraw()`` 来强制Widget刷新并重新绘制UI控件，这个方法会触发Widget重新渲染，确保UI控件根据最新数据显示。

<br>

## Q6：如何解决自定义的UI控件被和平原生控件遮挡的问题

由于和平原生控件分散在不同的控件蓝图中，ZOrder属性无法解决跨控件蓝图的层级问题，推荐使用控件锚点的方案，使用锚点可以将自定义的控件添加到指定层级，结合控件布局实现更精准的挂载效果，解决被和平控件遮挡的问题，具体操作请查看 [和平控件锚点](https://developer.gp.qq.com/wikieditor/#/catalog/20097)。

<br>

## Q7：如何隐藏主界面上的和平精英原生控件

目前部分常用的控件已接入WidgetLayout蓝图，且WidgetLayout蓝图已作为玩家默认的主界面配置，从UGCPlayerController蓝图中通过MainUIClass属性可定位到该蓝图，通过修改蓝图中各控件的可视性来实现显隐控制，也允许脚本中调用 [HideWidget](https://developer.gp.qq.com/api/#/searchContent/UGCWidgetManagerSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2FUI%20%E7%95%8C%E9%9D%A2%2FUGCWidgetManagerSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCWidgetManagerSystem&autoJump=HideWidget) 和 [ShowWidget](https://developer.gp.qq.com/api/#/searchContent/UGCWidgetManagerSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2FUI%20%E7%95%8C%E9%9D%A2%2FUGCWidgetManagerSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCWidgetManagerSystem&autoJump=ShowWidget) 动态控制显隐；对于未接入WidgetLayout蓝图的控件，只支持脚本动态控制显隐，需要查询主界面控件路径，拿到控件的实例对象，调用 [AddAdvancedCollapsedCount](https://developer.gp.qq.com/api/#/searchContent/UWidget?classDetailShow=true&path=class%2Fdetail%2FOthers%2FUWidget.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UWidget&autoJump=AddAdvancedCollapsedCount) 和 [SubAdvancedCollapsedCount](https://developer.gp.qq.com/api/#/searchContent/UWidget?classDetailShow=true&path=class%2Fdetail%2FOthers%2FUWidget.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UWidget&autoJump=SubAdvancedCollapsedCount) 实现显隐控制。
