# 视频播放器组件

在绿洲启元编辑器中，开发者们可以使用视频播放器组件在场景内播放视频，下面会详细介绍如何使用视频播放器组件。

<br>

## 使用流程

1. 创建蓝图，添加视频播放器组件和用来显示播放内容的静态网格体

2. 准备好要播放的媒体，存入或导入相应位置

3. 实例化蓝图，配置视频播放器组件属性

<br>

视频源的配置：在`UGCMovie Path`一栏填写格式为`Movies/[视频文件名]`的路径，如`Movies/LoginMovieBG.mp4`。

![企业微信截图_1686834934242.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686834934242.png)

音频源的配置：在`MovieSound`一栏中选择导入的音频即可。

![企业微信截图_16868349436779.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868349436779.png)

`MediaTextureBase`：在`Media Texture Base`一栏中选择`UGCMediaTexture0`、`UGCMediaTexture1`中的任意一个（请勿选择其他MediaTexture）。

`MediaPlayerTemplate`：在`Media Player Template`一栏中选择`UGCMediaPlayer0`、`UGCMediaPlayer1`中的任意一个（请勿选择其他MediaPlayer），这里选择的MediaPlayer编号需要与`MediaTextureBase`中对应，如`UGCMediaTexture0`对应`UGCMediaPlayer0`。

<span style="color:red">**注意：出于性能考虑，同一场景中不应出现多于2个带有视频播放器组件的蓝图。**</span>

## 其他参数含义

**Stand by Materials：** 当未在播放视频时，会默认显示此处的材质，数组为空时，会显示VideoMesh的默认材质

![企业微信截图_1686834962763.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686834962763.png)

**Static Image Materials：** 部分机型不支持播放视频，当机型不支持视频时，会显示此材质，数组为空时，会显示VideoMesh的默认材质

![企业微信截图_16868349743394.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868349743394.png)

**Activate Distance：** 触发距离，超过此距离视频将不再播放，单位为`厘米`

![企业微信截图_168683498291.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_168683498291.png)

**Loop Media：** 是否循环播放媒体，勾选则可以循环播放视频

![企业微信截图_16868349902425.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868349902425.png)

**Auto Active：** 是否自动激活组件，勾选则可以让媒体播放器在玩家进入触发距离时自动播放视频，否则需要通过`[视频播放器组件名]:InitPlayBack()`来控制组件的播放，且将`Activate Type`设置成`DVPCAT MANUAL`，示例代码如下：

```lua
---@class BP_MediaPlayer_C:AActor
---@field Plane UStaticMeshComponent
---@field BP_UGCVideoPlayer UBP_UGCVideoPlayer_C
---@field DefaultSceneRoot USceneComponent
--Edit Below--

local BP_MediaPlayer = {}; 

function BP_MediaPlayer:ReceiveBeginPlay()
    self.SuperClass.ReceiveBeginPlay(self);
    self.BP_UGCVideoPlayer:InitPlayBack() -- 在BeginPlay时播放视频
end

function BP_MediaPlayer:ReceiveEndPlay()
    self.SuperClass.ReceiveEndPlay(self); 
    self.BP_UGCVideoPlayer:StopPlayBack() -- 在EndPlay时停止视频

end

return BP_MediaPlayer;
```

**Activate Type：** 播放器组件类型，状态为`DVPCAT BY RANGE`时，需要勾选`Auto Active`方可播放，状态为`DVPCAT MANUAL`时，需要通过代码来控制播放

![企业微信截图_16868349999564.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868349999564.png)

**Path Type：** 视频文件所在的目录，枚举，默认为`UGC`，选择`Trunk`则可以播放和平精英包内的视频
