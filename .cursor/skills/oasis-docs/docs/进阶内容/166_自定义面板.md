#  自定义面板

- 自定义面板是《和平精英》非常重要且好用的功能，玩家自定义面板的比例非常高，如果你的玩法有自己新加的按钮/显示UI等，强烈建议接入和平的自定义面板系统，以供玩家自定义自己的UI位置、大小、透明度等

#### 2.2 在特定目录下创建用于自定义面板中的控件蓝图并且编辑控件蓝图，并且起名为UGCLayout_BP，修改父类为UAEUser Widget

- 在你的项目/Asset目录下创建控件蓝图，并且起名为UGCLayout_BP
![企业微信截图_16868329039044.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868329039044.png)
- 编辑该蓝图，注意层级结构需要与3.1中的实际生效的控件蓝图结构、锚点、位置等一致，并且Customize Canvas Panel BP的Elem Type需要设置成与3.1中一样;Customize Canvas Panel BP和Border 的显示为Visible
![企业微信截图_16868329162392.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868329162392.png)
- <span style="color: #EE0000"> **【注意】:建议用直接复制的方式，保证锚点、层级等一致，并且UGCLayout的按钮不需要Button，只需要对应的视觉样式图标** </span>

#### 2.3 在工程lua合适的位置添加关键代码

```lua
function UGCWidgetManagerSystem.AddChildToTochButton(Widget)
```

<br>
如示例工程所示，在UGCGameState.lua文件中<br>

```lua
---@class UGCGameState_C:BP_UGCGameState_C
--Edit Below--
local UGCGameState = {}; 
UGCGameState.MainUIClassPath = UGCMapInfoLib.GetRootLongPackagePath().. "Asset/testMainUI.testMainUI_C";     --test按钮所在控件蓝图路径MainUIClassPath
 function UGCGameState:ReceiveBeginPlay()
    local PlayerController = GameplayStatics.GetPlayerController(self, 0); --拿到客户端的PlayerController

    local MainUIClass = UE.LoadClass(UGCGameState.MainUIClassPath);  --加载MainUIClassPath的类
    local MainUI = UserWidget.NewWidgetObjectBP(PlayerController, MainUIClass); --新建MainUIClass的对象
    --MainUI:AddToViewport();
    UGCWidgetManagerSystem.AddChildToTochButton(MainUI); --关键函数，应用到和平主界面上并且读取自定义面板中玩家设置的UI信息
 end

return UGCGameState;
```
