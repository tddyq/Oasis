#  语音系统

### 功能介绍

和平语音系统为和平和绿洲玩法内置功能，主要用于局内玩家语音对话。
该功能进入游戏后自动开启，频道一般分为队伍语音和全局语音，队伍语音会按照进入游戏时的组队关系分配，如果开发者在游戏内对队伍进行了重新分配，那么语音也需要相应的进行分配。

### 接口库

API文档：[UGCVoiceManagerSystem](https://developer.gp.qq.com/api/#/searchContent/UGCVoiceManagerSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2F%E8%AF%AD%E9%9F%B3%2FUGCVoiceManagerSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCVoiceManagerSystem)

### 使用说明

接口库中最常用的功能为UGCVoiceManagerSystem.JoinVoiceRoom，在队伍关系重新分配后，需要把同队的人分配到同一个房间，可以把玩家的队伍ID传入客户端，然后自行组合RoomKey，如TeamRoom_<span style="color: #EE0000">TeamID</span>。

如在TeamID的OnRep（属性同步）事件中，为玩家分配对应的队伍房间

### 示例代码

```lua
function UGCPlayerController:OnRep_UGCTeamID()				--UGCTeamID为开发者自己注册的同步参数
    local RoomName = "TeamVoiceRoom"..tostring(self.UGCTeamID)
    UGCVoiceManagerSystem.JoinVoiceRoom(RoomName)
end
```
