#  开启和平精英集成系统功能

在绿洲启元编辑器中，可以使用部分和平精英封装的集成系统功能，下面简单介绍如何在编辑器内开启这些功能。

<br>

## 操作步骤

1. 编辑器的GameMode蓝图，这里的GameMode使用的是UGCGameMode

![企业微信截图_16868329684052.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868329684052.png)

2. 在 Component Manager 分类中的Component Configs Array中设置你想要开启的功能，例如下图中开启了航线功能

![企业微信截图_1686832981543.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686832981543.png)

3. 在Data Manager分类中的Data Sources Array中设置开启的功能的参数配置，例如下图中配置了航线使用的飞机和飞机AI

![企业微信截图_16868329935279.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868329935279.png)

4. 接下来可以通过设置功能的相关配置参数使功能正式生效，加速你的游戏开发

<br>

## 常见功能

绿洲启元编辑器封装了众多和平精英的集成系统功能，常见的配置如下：

| 系统  | SubSystemId | SubSystemClass | Data Source |
| -| -| -| -|
| 航线 | ESS_ClassicPlane | PlaneComponent | GMData Source Air Route |
| 空投 | ESS_AirDrop | AirDropComponent | GMData Source Air Drop |
| 信号圈 | ESS_Circle | CircleMgrComponent | GMData Source Poison Circle |
