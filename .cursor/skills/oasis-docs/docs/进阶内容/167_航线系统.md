## 实战演练

1.打开GameMode蓝图，并修改航线系统相关配置，可详见 开启和平精英功能wiki：
![企业微信截图_16868331064374.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868331064374.png)
2.配置航线系统相关参数：
![企业微信截图_16868331146166.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868331146166.png)
3.在Lua中使用API调用接口开始飞行：

```
UGCBlueprintFunctionLibrary.StartAirRoute(UGCGameSystem.GameState);
```

## 主要参数含义

![企业微信截图_16868331261140.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868331261140.png)

1. 根据Map Center，Radius 1 ，Radius 2，Radius 3 确定Circle 1 ，Circle 2 和 Circle 3.<br>
2. 任取 在Circle 1上的 PointA 和 在Circle 2上的 PointB，确定航线 Air Route<br>
3. Air Route 和 Circle 2 的交点 为 跳伞点 和 强制跳伞点<br>

## 示例工程

见附件，在示例工程中，只要 角色 和 Cube Actor重叠，就会触发 Cube.lua 中的 OnBeginOverlap，从而开始航线飞行：

```
function HelloWorld:OnBeginOverlap(OverlappedComp, Other, OtherComp, OtherBodyIndex, bFromSweep, SweepResult)
    UGCBlueprintFunctionLibrary.StartAirRoute(UGCGameSystem.GameState);
end
```
