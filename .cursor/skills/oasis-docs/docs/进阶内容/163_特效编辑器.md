# 特效编辑器

特效编辑器支持从特效模板中创建特效，并且根据您的需要，进行修改。

## 基本概念

### 粒子系统介绍

在游戏场景中存在的粒子/特效，实际上是一个粒子系统（Particle System），一个粒子系统中包括多个粒子发射器（Particle Emmitter），每一个粒子发射器包含多个模块。

![image.3 (1).png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/CM5tXimage.3%20(1).png)

以【网格粒子模板】的【径向】特效为例，它共包含五个粒子发射器。第一个为普通发射器，它控制地面贴图效果；第二个和第五个为爆发粒子发射器，它控制火星和沙砾的爆发效果；第三个和第四个为网格粒子发射器，它控制网格的生成效果。

|整体特效|第一个发射器|第二个发射器|第三个发射器|第四个发射器|第五个发射器|
|:-:|:-:|:-:|:-:|:-:|:-:|
|![生长.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/mb6S3%E7%94%9F%E9%95%BF.gif)|![第一层.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Ec7Gc%E7%AC%AC%E4%B8%80%E5%B1%82.gif)|![第二层.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/WXccP%E7%AC%AC%E4%BA%8C%E5%B1%82.gif)|![第三层.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/gw9QT%E7%AC%AC%E4%B8%89%E5%B1%82.gif)|![第四层.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/txzb2%E7%AC%AC%E5%9B%9B%E5%B1%82.gif)|![第五层.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/zCzfh%E7%AC%AC%E4%BA%94%E5%B1%82.gif)|

### 粒子发射器介绍

特效编辑器提供了多种粒子发射器，开发者可以通过组装发射器，实现完整、绚丽的特效。

|发射器类型|效果|说明|
|:-:|:-:|-|
|普通粒子发射器|![Simple.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ozQRiSimple.gif)|这是最常见、最基础的粒子发射器类型，也被称为“精灵”发射器。|
|爆发粒子发射器|![bu.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/FK6hIbu.gif)|“爆发”其实是一种发射模式（Burst），它通常与普通粒子发射器结合使用。|
|拖尾粒子发射器|![to.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/V73vFto.gif)|用于创建带状或轨迹类的效果。|
|射线粒子发射器|![se.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/wmFUbse.gif)|用于模拟激光、闪电、能量射线等线性效果。|
|网格粒子发射器|![me (1).gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/QcbAhme%20(1).gif)|与普通发射器使用2D纹理不同，网格体发射器使用3D静态网格体（Static Mesh） 作为粒子。|

### 特效浏览区

在特效浏览区的【Asset Tree】可以查看选择已创建的特效，在【Template】可单击选择特效模板，然后点击“选择模板创建”，进行特效创建。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/xV7YKimage.png)

---

### 特效视口区

双击打开特效文件后，可在【特效视口区】实时预览粒子效果，通过左上角的【视图】和【时间】控件，可以切换浏览模式与调整播放控制，关于视口的操作方法，请参阅 [编辑器视口](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/editor-viewports?application_version=4.27) 。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/mzBHaimage.png)

在【视图】控件的天空盒选项中，可以切换不同的天空盒样式，改变预览特效时的天空盒背景。

![PixPin_2026-01-14_11-17-23-ezgif.com-video-to-gif-converter.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Qx0JXPixPin_2026-01-14_11-17-23-ezgif.com-video-to-gif-converter.gif)

### 细节编辑区

一个完整的特效存在多个粒子发射器，一个发射器存在多个模块。打开特效后，可在【细节编辑区】对特效进行编辑调整，实现定制化效果。

![特效编辑器wiki绘图-细节编辑区CG34 的副本.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ya4ZP%E7%89%B9%E6%95%88%E7%BC%96%E8%BE%91%E5%99%A8wiki%E7%BB%98%E5%9B%BE-%E7%BB%86%E8%8A%82%E7%BC%96%E8%BE%91%E5%8C%BACG34%20%E7%9A%84%E5%89%AF%E6%9C%AC.drawio.png)

在左侧选择发射器，可查看发射器包含的模块或模块的详细配置信息。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/NIxSmimage.png)

点击发射器或模块的勾选按钮，可选择是否启用对应发射器或模块，启用后特效会显示发射器或发射器会显示模块效果。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/58KIFimage.png)

点击发射器最右侧的加号按钮，可选择添加发射器（AddEmitter）和管理发射器（重命名/删除/导出）。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/3q9W7image.png)

|操作名称|操作说明|
|:-:|-|
|重命名发射器|修改当前发射器的名称|
|删除发射器|删除当前发射器|
|导出发射器|将当前发射器导入到指定特效中<br>![导出一个.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/PM4qQ%E5%AF%BC%E5%87%BA%E4%B8%80%E4%B8%AA.gif)|
|全部导出|将当前特效的所有发射器导入到指定特效中<br>![ezgif-73aaf447af7a41bf.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ONL6gezgif-73aaf447af7a41bf.gif)|
|AddEmitter|新增发射器|
|新建Sprite|创建普通粒子发射器|
|新建Burst|创建爆发粒子发射器|
|新建Beam|创建射线粒子发射器|
|新建Mesh|创建网格粒子发射器|
|新建Ribbon|创建拖尾粒子发射器|

## 粒子模块介绍

特效编辑器中，每个发射器包含多个不同类型的粒子模块，可通过启用并修改粒子模块来改变发射器的表现效果。

### 基础设置

基础设置模块用于控制粒子发射的基本行为。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/tHQU2image.png)

|属性名称|说明|
|:-:|-|
|发射率|设置发射器在播放期间每秒产生的粒子数量|
|爆发数量|设置单次爆发事件中瞬时生成的粒子数量|
|爆发次数|设置发射器单次播放周期内触发的爆发事件总数|
|持续时间|设置发射器单次播放的总时长，单位：秒|

---

### 材质

材质模块用于定义粒子渲染所使用的材质资源及其动画属性。

![ScreenShot_2025-11-12_150701_783.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/qteGLScreenShot_2025-11-12_150701_783.png)

|属性名称|说明|
|:-:|-|
|发射器材质|指定用于渲染粒子外观的材质资源|-|
|UV数量|此参数定义发射器材质在水平(X)和垂直(Y)方向上的划分数量，粒子将按从左到右、从上到下的顺序逐帧播放|

![UV数量对比.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/HaQuHUV%E6%95%B0%E9%87%8F%E5%AF%B9%E6%AF%94.gif)

### 生命内颜色

生命内颜色模块用于控制粒子在发射过程中的颜色与透明度变化。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/maNDCimage.png)

颜色部分的配置说明如下：

|属性名称|说明|
|:-:|-|
|类型|设置粒子颜色的变化方式<br> • 常数：粒子颜色在整个生命周期中始终固定不变<br> • 曲线：粒子颜色在生命周期内从开始颜色渐变到结束颜色|
|值|设定固定的颜色，【类型】选择“常数”时生效|
|开始|设定粒子在开始生命周期的颜色，【类型】选择“曲线”时生效|
|结束|设定粒子在结束生命周期的颜色，【类型】选择“曲线”时生效|

Alpha部分的配置说明如下：

|属性名称|说明|
|:-:|-|
|类型|设置粒子透明度的变化方式<br> • 常数：粒子在整个生命周期中保持固定透明度<br> • 曲线：粒子透明度在生命周期内从开始值渐变到结束值|
|值|设定粒子固定的颜色透明度，值越低越透明，【类型】选择“常数”时生效|
|开始|设定粒子在开始生命周期的颜色透明度，值越低越透明，【类型】选择“曲线”时生效|
|结束|设定粒子在结束生命周期的颜色透明度，值越低越透明，【类型】选择“曲线”时生效|

### 生命内颜色比例

生命内颜色比例模块用于控制粒子在发射过程中的颜色与透明度的叠加。粒子在生命周期中，其颜色与透明度将根据本模块的设置进行叠加计算。叠加方式为乘法混合，例如黄色叠加蓝色将得到绿色效果。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/7QAUXimage.png)

颜色部分的配置说明如下：

|属性名称|说明|
|:-:|-|
|类型|设置叠加颜色的变化模式<br> • 常数：粒子在整个生命周期中保持固定的叠加颜色<br> • 曲线：叠加颜色在生命周期内从开始色渐变到结束色|
|值|设定固定的叠加颜色，【类型】选择“常数”时生效|
|开始|设定粒子生命周期起始时的叠加颜色，【类型】选择“曲线”时生效|
|结束|设定粒子生命周期结束时的叠加颜色，【类型】选择“曲线”时生效|

Alpha部分的配置说明如下：

|属性名称|说明|
|:-:|-|
|类型|设置叠加透明度的变化模式<br> • 常数：粒子在整个生命周期中保持固定的叠加透明度<br> • 曲线：叠加透明度在生命周期内从开始值渐变到结束值|
|值|设定粒子固定的叠加颜色透明度，值越低越透明，【类型】选择“常数”时生效|
|开始|设定粒子生命周期起始时叠加颜色的透明度，值越低越透明，【类型】选择“曲线”时生效|
|结束|设定粒子生命周期结束时叠加颜色的透明度，值越低越透明，【类型】选择“曲线”时生效|

### 大小随生命周期

该模块用于控制粒子在运动过程中的大小变化。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/qIaoAimage.png)

|属性名称|说明|
|:-:|-|
|类型|设置粒子大小的变化模式<br>• 常数：粒子大小在整个生命周期内保持固定，不发生变化<br>• 随机：粒子的大小会在设定的范围内，每帧进行随机变化<br>• 曲线：粒子大小在其生命周期内，从开始倍率平滑过渡到结束倍率。|
|值|设定固定的大小倍率，【类型】选择“常数”时生效|

### 摄像机偏移

摄像机偏移用于微调粒子与摄像机之间相对位置，填入的数值越大，粒子离摄像机越近。通常用于以下两种情况：
- 避免摄像机穿透：当摄像机靠近或移动通过粒子效果（如雾气、灰尘）时，过近的距离会导致粒子不自然地放大或出现视觉瑕疵。通过设置适当的 Z 轴正偏移值（例如设置为 50，表示粒子沿摄像机前方偏移 50 个单位），可以确保粒子始终与摄像机保持一定距离，维持视觉上的合理性。
- 创建UI关联特效：对于需要固定在屏幕特定位置的粒子元素（如角色状态图标、对话气泡等），可通过计算并设置合适的偏移量，使粒子效果“附着”于屏幕指定区域，不受摄像机移动或旋转的影响，实现 UI 级别的视觉表现。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/fTi8Aimage.png)

|属性名称|说明|
|:-:|-|
|类型|设置摄像机偏移的方式<br>• 常数：粒子的摄像机偏移值恒定不变<br>• 曲线：粒子的摄像机偏移值在其生命周期内，从开始值平滑过渡到结束值|
|值|设定粒子固定的摄像机偏移值，【类型】选择“常数”时生效|
|开始|粒子在生命周期初始时刻的摄像机偏移值，【类型】选择“曲线”时生效|
|结束|粒子在生命周期结束时刻的摄像机偏移值，【类型】选择“曲线”时生效|
