# 聊天系统

# 功能介绍

和平精英的聊天系统是和平精英和绿洲玩法内置功能，主要用于局内玩家文字交流。
<br>该功能由开发者在工程设置中选择是否开启，若开启聊天功能，将在UI内显示聊天入口，点击可以打开聊天界面。

勾选![企业微信截图_16868341324308.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868341324308.png)将开启聊天界面中的全房间聊天频道。聊天频道中包含了当前游戏房间内的所有玩家。

## 私聊频道

若希望房间内的玩家可以进行私聊，那么就勾选![企业微信截图_16868341885752.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868341885752.png)开启私聊频道。频道内玩家可以与房间内的玩家一对一聊天。

---

## 队伍/阵营聊天频道

如果玩法中，对玩家进行队伍或者阵营的划分，并且希望队伍或者阵营内部进行文字交流，那么就可以勾选![企业微信截图_1686834199156.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1686834199156.png)开启队伍聊天频道。
聊天频道开启后，需要将玩家的队伍信息通过接口通知聊天系统，这样玩家才能加入指定的队伍聊天。

```
-- 将指定玩家加入指定队伍的聊天频道
-- @param PlayerKey 玩家的PlayerKey值
-- @param CampID 队伍的ID，玩法自行管理分配，传入0为无阵营
UGCMessageSystem.JoinCampMessageChannel(int PlayerKey, int CampID)
```

## 系统频道

系统频道用来将系统消息发送给玩家，玩家并不能在系统频道进行聊天。勾选![企业微信截图_16868342165004.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868342165004.png)后开启系统频道，然后玩法可以使用接口将期望发送的系统消息发送至聊天界面的系统频道内。

```
-- 将系统消息发送给所有玩家
-- @param MessageTag 系统消息标签，如“系统通知”
-- @param MessageContent 系统消息内容
UGCMessageSystem.SendSystemMessageToAll(String MessageTag, String MessageContent)

-- 将系统消息发送给指定玩家
-- @param PlayerKey 玩家的PlayerKey值
-- @param MessageTag 系统消息标签，如“系统通知”
-- @param MessageContent 系统消息内容
UGCMessageSystem.SendSystemMessageToPlayer(int PlayerKey, String MessageTag, String MessageContent)
```

## 聊天气泡

若想开启聊天内容发送后，将聊天内容显示在玩家的头顶气泡内，则需要添加聊天气泡功能。打开玩家控制的Pawn，添加组件TopChatBubble，并配置组件参数。
组件参数分别为：

1. 保留时间：气泡显示时间，之后将消失，直到再次发送聊天内容；
2. 显示位置：气泡的补偿位置，可以用于调整气泡位置；
