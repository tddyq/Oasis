# 电磁干扰区系统

电磁干扰区是游戏内的战术环境机制，可在指定区域生成持续性的电磁干扰效果，该效果可导致载具熄火、瞄具准星消失，并使玩家屏幕出现视觉花屏干扰，开发者可以利用电磁干扰区系统，剥夺范围内玩家的机动性与作战效能，丰富游戏的战术选择和策略深度。

<br>

## 电磁干扰区流程

电磁干扰区系统按照以下流程执行相应的阶段行为：

![信号圈系统-电磁干扰区流程.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/R3yaT%E4%BF%A1%E5%8F%B7%E5%9C%88%E7%B3%BB%E7%BB%9F-%E7%94%B5%E7%A3%81%E5%B9%B2%E6%89%B0%E5%8C%BA%E6%B5%81%E7%A8%8B.drawio.png)

1. 生成指令下达后，电磁干扰区不会立即显现，而是进入一段准备时间，区域内暂无电磁干扰效果
2. 准备时间结束后，电磁干扰区出现并生效，指定区域出现电磁干扰效果，同时小地图会显示电磁干扰区的位置与范围
3. 当设定的持续时长结束时，电磁干扰区效果自动消失，受影响区域内的一切功能将恢复正常

<br>

## 配置依赖

### BP_EMPZoneControlComp

打开 ``UGCGameMode`` , 添加组件 ``BP_EMPZoneControlComp`` ,该组件用于管理与处理电磁干扰区的配置和逻辑。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/a1vS8image.png)

并在该组件中，属性【EMPZoneClass】选择 ``BP_UGCEMPZoneActor`` 。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/bXXTbimage.png)

### GamePart

在编辑器主界面的工具栏，点击进入【玩法通用设置】。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/sPaCZimage.png)

点击【Active Game Part Configs】右侧的加号，添加GamePart模块，然后在新增的数组元素选择需要启用的GamePart模块。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Fk2TLimage.png)

选择启用 ``GP_EMPZone`` 模块，电磁干扰区的信息从该模块配置。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/0ytErimage.png)

<br>

## 电磁干扰区配置

进入【玩法通用设置】，从【Active Game Part Configs】找到 ``GP_EMPZone`` 模块，点击【EMPZone Configs】右侧的加号，添加新的电磁干扰区的配置项。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/DDdg1image.png)

电磁干扰区的配置项以及说明如下：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/nlKTAimage.png)

|属性名称|说明|
|:-:|-|
|GenerationTime|电磁干扰区的出现时间|
|StartTime|电磁干扰区的持续时长|
|EffectRadius|电磁干扰区作用半径，单位：厘米|

<br>

## 生成电磁干扰区

在玩法脚本中可以通过调用 [``GenerateElectromagneticArea``](https://developer.gp.qq.com/api/#/searchContent/UGCEMPZoneSystem?classDetailShow=true&path=class%2Fdetail%2FOthers%2FUGCEMPZoneSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCEMPZoneSystem&autoJump=GenerateElectromagneticArea) 在指定的位置生成电磁干扰区，生成后即自动启动电磁干扰区的流程，调用 [``DestroyElectromagneticArea``](https://developer.gp.qq.com/api/#/searchContent/UGCEMPZoneSystem?classDetailShow=true&path=class%2Fdetail%2FOthers%2FUGCEMPZoneSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCEMPZoneSystem&autoJump=DestroyElectromagneticArea) 可以关闭电磁干扰区域。

![生成电磁干扰区.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/y7f2t%E7%94%9F%E6%88%90%E7%94%B5%E7%A3%81%E5%B9%B2%E6%89%B0%E5%8C%BA.gif)
