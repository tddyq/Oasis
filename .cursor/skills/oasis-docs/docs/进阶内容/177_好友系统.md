#  好友系统

## 功能介绍：

好友系统是和平集成的社交系统，开发者可以通过在适当的时机让玩家相互添加好友，增加玩家组队游玩热情从而提高玩法留存率。
最常见的添加好友的渠道是在结算界面增加添加好友的按钮，部分更重社交的游戏还可以在其他位置也添加。

## 相关接口

|接口 |描述 |适用范围 |
| ------ | ------ | ------ |
|FriendSystem.IsMyFriend(UID) |返回玩家是否已是好友 | 客户端 |
|FriendSystem.AddFriend(UID, BP_ENUM_ADD_FRIEND_FROM_BATTLE_RESULT, ContentIndex) |申请添加好友，其中ContentIndex，传入7为对男性用语，传入8为对女性用语 | 客户端 |

## 示例代码

```
--添加好友按钮按下
function Button_AddFriend_OnClicked()
    --发送好友申请
    if PlayerSex == 1 then
        FriendSystem.AddFriend(UID, BP_ENUM_ADD_FRIEND_FROM_BATTLE_RESULT, 7); -- 7是在好友申请配置表配置的男性默认语
    else
        FriendSystem.AddFriend(UID, BP_ENUM_ADD_FRIEND_FROM_BATTLE_RESULT, 8); -- 8是在好友申请配置表配置的女性默认语
    end
    PopUpNoticeUI.ShowFastNoticeQueue(string.format("已向%s发送好友申请消息", self.PlayerResultData.PlayerName));
end
```

PS：具体可以参照躲猫猫的结算UI逻辑，有现成的案例。
