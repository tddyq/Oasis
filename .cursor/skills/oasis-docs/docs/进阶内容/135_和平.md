#  和平

# 通用

## 打印日志（UGC_OutputLog）

可以当前行为树相关信息到日志中

## 等待（UGC_Wait）

**等待（UGC_Wait）** 任务节点可以在行为树中使用，使树在此节点上等待，直至指定的 **等待时间（Wait Time）** 结束。

## 修改黑板变量值（UGC_ModifyBBValue）

修改指定黑板变量的变量值

---

# 角色

## 对准朝向（UGC_SetFocus）

按照一定的速度转向目标。

## 获取随机点（UGC_GetRandomLocation）

获得指定目标附近的随机点。

## 射击(UGC_Shooting)

射击

## 释放技能（UGC_CastSkill）

释放指定类型的技能。
注意事项：该节点通过技能实现，需要pawn上配置对应的技能

## 寻找指定类型Actor（UGC_GetActorOfClass）

寻找周围指定类型的Actor

---

# 怪物

## 怪物移动(UGC_MonsterMove)

怪物移动到指定目标位置

## 装备武器(UGC_EquipWeapon)

怪物装备武器，不走背包

# 主角

## 主角移动(UGC_PlayerMove)

控制主角移动到指定位置

## 刷物品(UGC_SpawnItem)

生成道具

## 近战攻击(UGC_MeleeAttack)

主角释放近战攻击
注意事项：该节点通过技能实现，需要pawn上配置对应的技能

## 设置角色状态(UGC_SetPawnState)

设置主角的状态，比如游泳，战斗，拾取等等

## 设置角色姿势(UGC_SetPose)

设置主角的姿势，比如站立，跳跃，下蹲等

## 吃药(UGC_TakeMedicine)

注意事项：该节点通过技能实现，需要pawn上配置对应的技能

| 属性 | 描述 |
| :--- | :---: |
| **药品类型** | 绷带，止痛药等 |
| **Ignore Restart Self** | 启用后，当选择执行的任务已在运行时，任务搜索将被放弃 |
| **Node Name** | 节点在行为树图表中显示的名称 |
