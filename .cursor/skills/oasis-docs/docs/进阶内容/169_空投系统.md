# 空投系统

空投系统是游戏内重要的物资投放机制，可在指定位置生成包含各类物资的空投箱，为玩家提供额外的战略资源和战术选择，开发者可灵活配置空投箱内的物资内容，利用空投系统，丰富游戏过程中的资源争夺与策略博弈。

<br>

## 配置依赖

在编辑器主界面的工具栏，点击进入【玩法通用设置】。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/sPaCZimage.png)

点击【Active Game Part Configs】右侧的加号，添加GamePart模块，然后在新增的数组元素选择需要启用的GamePart模块。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Fk2TLimage.png)

选择启用 ``GP_AirDropManager`` 模块，空投系统的物资信息从该模块配置。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/79277image.png)

<br>

## 空投配置

进入【玩法通用设置】，从【Active Game Part Configs】找到 ``GP_AirDropManager`` 模块，空投的配置项以及说明如下：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/aCqI0image.png)

|属性名称|说明|
|:-:|-|
|ID|标识空投配置的唯一的ID|
|LifeSpan|空投从生成到消失的时间，单位：秒|
|ItemConfig|配置该空投包含的物资信息|
|物资配置模式|空投物资的配置方式，可选：<br>• 物资ID：直接指定物品ID与数量<br>• 掉落表：根据 [掉落表](https://developer.gp.qq.com/wikieditor/#/catalog/20058?autoJump=%E6%8E%89%E8%90%BD%E8%A1%A8) 中的掉落项配置物资<br>• 掉落组表：根据 [掉落组表](https://developer.gp.qq.com/wikieditor/#/catalog/20058?autoJump=%E6%8E%89%E8%90%BD%E7%BB%84%E8%A1%A8) 中的掉落组配置物资|
|物资ID|填写对应的物品ID，【物资配置模式】选择“物资ID”时生效|
|物资数量|填写物品的生成数量，【物资配置模式】选择“物资ID”时生效|
|掉落ID|填写掉落表中对应的掉落ID，【物资配置模式】选择“掉落表”时生效|
|掉落组ID|填写掉落组表对应的掉落组ID，【物资配置模式】选择“掉落组表”时生效|

<br>

## 生成空投

在玩法脚本中可以通过调用 [``GenerateAirDrop``](https://developer.gp.qq.com/api/#/searchContent/UGCAirDropManagerSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2F%E7%8E%A9%E6%B3%95%E8%A7%84%E5%88%99%2FUGCAirDropManagerSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCAirDropManagerSystem&autoJump=GenerateAirDrop) 在指定的位置生成空投。

![PixPin_2025-12-22_17-00-38-ezgif.com-video-to-gif-converter.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ixrb2PixPin_2025-12-22_17-00-38-ezgif.com-video-to-gif-converter.gif)
