# 轰炸区系统

轰炸区是游戏内的战术环境机制，可对指定区域持续投放炸弹并造成打击，开发者可灵活配置轰炸区的范围、时长、伤害参数等属性，利用轰炸区系统实现对特定区域的控制或高难度挑战，从而丰富游戏的战术维度和策略深度。

<br>

## 轰炸区流程

轰炸区系统按照以下流程执行相应的阶段行为：

![信号圈系统-轰炸区流程.drawio.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/QgVMP%E4%BF%A1%E5%8F%B7%E5%9C%88%E7%B3%BB%E7%BB%9F-%E8%BD%B0%E7%82%B8%E5%8C%BA%E6%B5%81%E7%A8%8B.drawio.png)

1. 生成指令下达后，轰炸区进入预警阶段，区域内暂未发生实际轰炸，小地图会以特定颜色显示轰炸区的位置与范围
2. 预警阶段结束后，轰炸区进入轰炸阶段，轰炸区内落下炸弹，小地图上的区域提示颜色将切换为另一种颜色
3. 设定的轰炸阶段结束时，所有轰炸行为立即停止，轰炸区效果完全消失，小地图上对应的区域提示也随之清除

<br>

## 配置依赖

在编辑器主界面的工具栏，点击进入【玩法通用设置】。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/sPaCZimage.png)

点击【Active Game Part Configs】右侧的加号，添加GamePart模块，然后在新增的数组元素选择需要启用的GamePart模块。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Fk2TLimage.png)

选择启用 ``GP_AirAttack`` 模块，轰炸区的信息从该模块配置。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/d58Ibimage.png)

<br>

## 轰炸区配置

进入【玩法通用设置】，从【Active Game Part Configs】找到 ``GP_AirAttack`` 模块，点击【Air Attack Configs】右侧的加号，添加新的轰炸区的配置项。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/teT0Vimage.png)

轰炸区的配置项以及说明如下：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/B7DBEimage.png)

|属性名称|说明|
|:-:|-|
|EscapeTime|轰炸区预警阶段时长，此阶段结束后进入轰炸阶段，单位：秒，范围：[10,180]|
|AttackLastingTime|轰炸区轰炸阶段时长，单位：秒，范围：[10,180]|
|AttackAreaRadius|轰炸区作用半径，单位：厘米|
|TotalBombCount|轰炸阶段炸弹投放总数，范围：[1,50]|
|BombMaxDamage|炸弹爆炸的中心范围伤害值，范围：[10,500]|
|BombMinDamage|炸弹爆炸边缘的最小伤害值，范围：[10,500]|
|BombRadius|单颗炸弹的爆炸影响半径，单位：厘米|
|CenterDamageRange|炸弹中心伤害范围，超出此范围伤害将衰减至最小伤害，单位：厘米|
|BombFallSpeed|炸弹下落速度（从1000米高空开始坠落），单位：厘米/秒，范围：[100,1000]|

下方【Air Attack】属性组中可以配置轰炸区在小地图的显示效果，相关配置项以及说明如下：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/cC5BUimage.png)

|属性名称|说明|
|:-:|-|
|Warning Color|预警阶段在小地图上显示的提示颜色|
|Bombing Color|轰炸阶段在小地图上显示的提示颜色|

## 生成轰炸区

在玩法脚本中可以通过调用 [``GenerateBombingArea``](https://developer.gp.qq.com/api/#/searchContent/UGCAirAttachSystem?classDetailShow=true&path=class%2Fdetail%2FOthers%2FUGCAirAttachSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCAirAttachSystem&autoJump=GenerateBombingArea) 在指定的位置生成轰炸区域，生成后即自动启动轰炸流程，调用 [``StopBombingArea``](https://developer.gp.qq.com/api/#/searchContent/UGCAirAttachSystem?classDetailShow=true&path=class%2Fdetail%2FOthers%2FUGCAirAttachSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCAirAttachSystem&autoJump=StopBombingArea) 可以终止轰炸流程。

![生成轰炸区.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/MQtKl%E7%94%9F%E6%88%90%E8%BD%B0%E7%82%B8%E5%8C%BA.gif)
