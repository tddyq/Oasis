#  局内观战

## 功能介绍

该功能主要用于游戏内玩家无法行动（胜利或者死亡）后，可以选择进入观战，避免玩家在等待过程中无聊。

## 相关接口

| 接口 | 描述 |  适用范围|
| ------ | ------ | ------ |
| UGCGameSystem.EnterSpectating(PlayerController) | 进入观战 | DS |
| UGCGameSystem.LeaveSpectating(PlayerController) | 退出观战 | DS | 
| UGCGameSystem.ChangeAllowOBPlayerKeys(PlayerController,PlayerKeyList)  | 设置可被观战玩家列表 | DS | 
| PlayerController:GetCurPawn() | 获取当前观战目标PlayerPawn | DS和客户端 | 
| PlayerController:OnSpectatorChange | 委托，观战目标切换后调用 | DS和客户端 |
| PlayerController:OnOverrideClickQuitSpectating | 委托，玩家点击退出观战后调用 | DS |
| PlayerController:OnRepIsSpectatingDelegate | 委托，玩家进入观战后调用 | DS和客户端 |

## 接口使用

UGCGameSystem.EnterSpectating  进入观战，随后调用UGCGameSystem.ChangeAllowOBPlayerKeys设置需要观战的目标列表，如果不设置观战目标列表，那么默认观战列表为队友。

## 示例代码

```
--观战所有玩家
local OBPlayerKeys = {}
local PlayerControllerList = UGCGameSystem.GetAllPlayerController()
for _, PlayerController in ipairs(PlayerControllerList) do
    if PlayerController and PlayerController ~= self then
        table.insert(OBPlayerKeys, PlayerController.PlayerKey)
    end
end
UGCGameSystem.EnterSpectating(self)   --进入观战
UGCGameSystem.ChangeAllowOBPlayerKeys(self, OBPlayerKeys)   --设置该玩家可以观战所有玩家。
```
