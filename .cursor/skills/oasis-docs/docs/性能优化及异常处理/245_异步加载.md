#  异步加载

## 介绍

---

- 使用UE.LoadObject同步加载会触发Flush导致帧率下降，造成短暂卡顿
- 非预加载的资源，建议使用附件中的代码，采用异步加载的方式加载资源
- <span style="color: #ff0000"> 异步加载要注意考虑异步加载之后，一些为空的异常情况，需要各类判空和使用IsValid判断</span>

<summary> <font face="楷体">示例：使用参考</font> </summary>

```lua
if UGCAsyncLoadTools ~= nil then
        UGCAsyncLoadTools:LoadObject(MeshPath, function(Asset)
            if self ~= nil and UE.IsValid(self) then
                self.Sphere:SetStaticMesh(Asset);
            end
        end)
    end
```
