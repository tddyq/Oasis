#  使用同模角色优化性能

## 功能介绍：

玩家同模是在玩法要求大量角色同屏，并且要求角色之间不需要辨识度的情况下的一个优化可选项。
效果为把所有角色设置为相同的模型，可以大幅度降低DrawCall。

## 相关接口

|接口 |描述 |适用范围 |
| ------ | ------ | ------ |
|PlayerPawn:getAvatarComponent() |获取Avatar组件 |服务端 |
|AvatarComponent:InitDefaultAvatarByResID(0, 401991, 0) | 把该角色变为同模角色 | 服务端 |

## 代码示例

```lua
local PlayerPawn = ScriptGameplayStatics.GetCharacterByPlayerKey(self, PlayerKey);
	local AvatarComponent = PlayerPawn:getAvatarComponent();
	AvatarComponent:InitDefaultAvatarByResID(0, 401991, 0)
```

## 注意事项

由于和平角色也会进行一个对自身模型的一个初始化流程，所以同模接口不易调用过早，需要等待PlayerPawn完全初始化完毕后调用，而且角色每次复活也会恢复和平角色，需要重新调用同模接口。
