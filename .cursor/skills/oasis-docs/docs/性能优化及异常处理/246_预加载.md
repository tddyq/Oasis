#  预加载

## 介绍

---

- 预加载的本质是空间换时间
- 资源加载方案选择的要点
  
  - 资源出现频率
  - 资源大小
  - 加载资源后，后续逻辑的复杂度
- <span style="color: #ff0000">  建议高频，必会使用的资源使用预加载，降低代码复杂度</span>
- <span style="color: #ff0000">  数量多，根据游玩情况不一定出现的内容使用异步加载，例如大量图标</span>
- GameState新建Object类型的列表，期望预加载的资源填入列表中
