#  网络裁剪

## 设置

---

- 设置Replication下Net Cull Distance Squared的值
  - 该值的单位为厘米的平方
  - 图中1600000000=40000厘米 * 40000厘米=400米 * 400米
  - 实际网络同步距离为400米
  - 即此Actor不会同步给距离超过自身400米的玩家

![LDD09.26fe18d3[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/LDD09.26fe18d3%5B1%5D.png)

- 网络裁剪距离越小，对网络和性能的压力越小
- 根据玩法的实际情况，设置合适的同步距离十分必要
- 距离设置了10米的演示情况
