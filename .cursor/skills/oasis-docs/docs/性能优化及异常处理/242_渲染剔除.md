#  渲染剔除

## 设置场景上的物件

---

- 使用地图-石头-海岛-海岛8号石头作为演示
- 把石头放置在场景中
- 拉近-拉远镜头观察石头是否渲染

<summary> <font face="楷体">摄像机距离石头很远，但石头依然渲染</font> </summary>

![LDD03.2bc4cef6[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/LDD03.2bc4cef6%5B1%5D.png)
![LDD04.9a476c39[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/LDD04.9a476c39%5B1%5D.png)

- 选中石头，设置以下内容
  - 勾选Override Culling Screen
  - 填写Culling Screen Size

![LDD05.77c596ba[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/LDD05.77c596ba%5B1%5D.png)
![LDD06.04081e23[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/LDD06.04081e23%5B1%5D.png)

<summary> <font face="楷体">摄像机距离石头一定距离，石头不渲染</font> </summary>

![LDD07.77e57027[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/LDD07.77e57027%5B1%5D.png)

- <span style="color: #ff0000"> 模型渲染面积占当前屏幕比例小于CullingScreenSize的值时，模型会不渲染</span>
- <span style="color: #ff0000"> 例如CullingScreenSize填0.1，代表模型渲染面积小于屏幕面击10%时，模型将不被渲染</span>

## 设置模型

---

- 自己导入的Mesh可以在Mesh中设置
- 和平资源为只读状态，只能放置在场景中修改
- 8号石头默认值0.01，所以默认设置下，很远都会被渲染
