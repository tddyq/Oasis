# 场景物件合批

数量在<span style="color:red;font-weight:bold;">5个及以上且缩放比例在0.5~2.0之间</span>的同类物件可以进行合批操作，合批后能有效减少DrawCall数量，提升客户端的渲染性能。

---

<br>

下面介绍物件合批的操作示例，例如关卡内有5个相同的石头Actor。
  
![001.6b131eea[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/001.6b131eea%5B1%5D.png)

选中这些石头Actor，右键点击“快速合并”。

![002.e084d9d4[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/002.e084d9d4%5B1%5D.png)

 合并后只有一个DrawCall。

![003.8cab52b9[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/003.8cab52b9%5B1%5D.png)

<br>

>**小技巧：**

如果场景物件的放置没有合理归类，可以利用以下操作快速批量选中同类物件：点击选中一个石头Actor，右键点击“选择”->“选择所有具有同样材质的项”。

![004.b7437d89[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/004.b7437d89%5B1%5D.png)

所有同类石头Actor将会被选中，然后再执行合批操作即可。

![005.7980fcca[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/005.7980fcca%5B1%5D.png)
