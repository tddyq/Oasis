# 通用伤害飘字

当游戏对象受到攻击时，通常以动态飘字的形式展示伤害数值，增强战斗视觉效果和反馈，伤害飘字功能提供对字体样式外观、飞行行为的自定义设置能力，同时支持特定伤害源的飘字效果定制，可应用于玩家角色与怪物。

## 启用伤害飘字

### 全局飘字开关

点击编辑器菜单栏【玩法通用设置】按钮，在“damage Number”类别下可进行飘字开关的管理。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/5w4huimage.png)

- 开启怪物受击伤害数字：怪物受击是否显示伤害飘字
- 开启玩家受击伤害数字：玩家受击是否显示伤害飘字
- 开启玩家治疗飘字：玩家受到治疗是否显示飘字
- 开启小怪治疗飘字：小怪受到治疗是否显示飘字

> 默认情况下，玩家角色没有开启飘字，开发者可按需启用

---

### 怪物飘字开关

玩法中通常有多种怪物蓝图，可能存在某些怪不产生飘字效果的需求，例如Boss怪，因此怪物蓝图单独提供了飘字开关。

打开怪物蓝图，在 ``TakeDamageLogicComp`` 组件的【细节】面板中找到 ``伤害冒字`` 属性进行设置。

> 全局飘字开关优先级高于怪物飘字开关

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/3obt7image.png)

## 设置飘字效果

伤害飘字的配置集成在 ``玩法通用设置`` 中，开发者只需简单修改参数，即可自定义飘字的效果。

点击编辑器菜单栏【玩法通用设置】按钮，打开 ``DA_GameModeGeneral`` 数据资产蓝图， 飘字的配置属性组在“Damage Number”类别下。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/zb9KIimage.png)

配置分为 ``默认伤害数字配置`` 和 ``特殊伤害数字配置`` 两部分，其中 ``默认伤害数字配置`` 提供基础飘字属性配置，属于全局默认的飘字效果；``特殊伤害数字配置`` 是基于基础飘字属性的扩展，结构为基于 [GameplayTag](https://developer.gp.qq.com/wikieditor/#/catalog/20102) 的伤害标签与飘字属性的键值组合对，适用于特定伤害的飘字效果。

### 基础飘字属性

![image.3.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/FhT08image.3.png)<br>

|属性名|属性说明|
|-|-|
|伤害数字的显示时间|飘字在屏幕上显示的持续时间（单位：秒）|
|伤害数字的飞行时间|飘字飞行到达最远点的时间（单位：秒），与飞行距离共同影响飘字的飞行速度，飞行距离固定的情况下飞行时间越少，飘字飞行的速度越快|
|伤害数字颜色|飘字的字体颜色|
|伤害数字描边颜色|飘字字体的描边颜色|
|伤害数字描边大小|飘字描边的宽度|
|淡入时间|飘字从出现到字体达到最大的总时间（单位：秒）|
|淡出时间|飘字从字体最大缩小至消失的总时间（单位：秒）|
|初始字体缩放系数范围|飘字初始大小的缩放倍率随机范围，X为最小值，Y为最大值；当X>=Y时，缩放倍率固定为X|
|伤害数字飞行距离缩放系数范围|飘字飞行距离的缩放倍率随机范围，X为最小值，Y为最大值；当X>=Y时，缩放倍率固定为X|
|伤害数字的初始位置水平偏移范围|基于受到伤害的位置，在水平方向上的随机偏移范围（单位：厘米），向右为正、向左为负；当X=Y时，水平偏移距离固定为X|
|伤害数字的初始位置垂直偏移范围|基于受到伤害的位置，在垂直方向上的随机偏移范围（单位：厘米），向上为正、向下为负；当X=Y时，垂直偏移距离固定为X|
|伤害数字的飞行角度范围|飘字飘出的随机角度范围，以正右方为起点，逆时针方向角度为负、顺时针方向角度为正；当X=Y时，飘出的角度固定为X|
|伤害数字前缀|飘字前的字符,支持大写英文字符和小写英文字符<br>![ScreenShot_2025-12-12_113120_962.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/fpcSrScreenShot_2025-12-12_113120_962.png)|
|伤害数字后缀|飘字后的字符,支持大写英文字符和小写英文字符<br>![39959fa5-76da-4bd9-af2f-0cb089d0775e.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Y6YaZ39959fa5-76da-4bd9-af2f-0cb089d0775e.png)|
|配置优先级|该配置的优先级，数值越大优先级越高，伤害有tag且存在对应tag的飘字配置时，使用对应tag的配置效果；否则使用默认飘字配置|
|触发大数字缩写的最小数字|大于该值则飘字数字会进行缩写，缩写单位支持K,M,B（默认值-1为关闭状态）<br>![ScreenShot_2025-12-12_152127_678.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/183xSScreenShot_2025-12-12_152127_678.png)|

> 飘字的字体大小和飞行距离的缩放倍率应合理设置，不合理的数值会导致飘字离开或超出屏幕范围

使用实体编辑器 [新建一个怪物蓝图](https://developer.gp.qq.com/wikieditor/#/catalog/20144?autoJump=%E5%88%9B%E5%BB%BA%E6%80%AA%E7%89%A9%E8%93%9D%E5%9B%BE) ，将该蓝图放置在关卡场景中实例化，调试游戏以查看飘字效果。

### 特定伤害的飘字效果

``Damage Number Config`` 属性配置的优先级高于默认飘字配置，允许为特定伤害类型定义特殊的飘字效果，目前支持伤害公式标签和技能伤害标签两种方式触发。

**技能伤害标签**

以践踏技能伤害为例创建一个特殊的飘字效果，打开 [技能编辑器](https://developer.gp.qq.com/wikieditor/#/catalog/20091) ，以 `践踏技能` 为模板新建一个技能，并且为该技能的伤害类型添加一个名为“Skill.SkillDamage”的标签，编译保存技能蓝图。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/GSl2vimage.png)
![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/OhqY5image.png)

打开 ``DA_GameModeGeneral`` 数据资产蓝图， 为 ``Damage Number Config`` 添加一组配置，其中 ``键`` 设置为践踏技能 ``造成伤害`` Task绑定的GameplayTag，即“Skill.SkillDamage”；同时为 ``值`` 配置不同的飘字属性，与默认飘字效果做区分。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/5fwMLimage.png)

再次启动调试，普通攻击的飘字效果为默认飘字属性配置，而当玩家触发践踏技能产生伤害时，飘字效果是“Skill.SkillDamage”标签对应的属性配置。

![20250611_152844-ezgif.com-video-to-gif-converter.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ohkgu20250611_152844-ezgif.com-video-to-gif-converter.gif)

**伤害公式标签**

[伤害公式](https://developer.gp.qq.com/wikieditor/#/catalog/20099?) 中为伤害返回结果添加额外的Tag信息也能触发特殊飘字效果，以“暴击伤害”为例，将暴击飘字的Tag添加到 ``ExtraResult.ResultTags``。

![伤害公式.gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/fNigQ%E4%BC%A4%E5%AE%B3%E5%85%AC%E5%BC%8F.gif)

``` lua
	 -- 是否暴击
    local IsCritical = (math.random() <= CauserCriticalChance)
    local CriticalRatio = IsCritical and ((1 + CauserCriticalRatio) * (1 - TargetCritDefenceRatio)) or 1
    local Crit = UGCGameplayTagSystem.RequestGameplayTag("UGC.DamageNumber.Crit")    --[NEW] 暴击tag

    print("[UGCGlobalDamageCalculation] Params: ".. "SkillAttack: ".. SkillAttack..", IsCritical: ".. tostring(IsCritical).. ", CriticalRatio: ".. CriticalRatio)
    if IsCritical and ExtraResult then
        local CritTag = UGCGameplayTagSystem.RequestGameplayTag("UGC.Damage.Result.Critical")
        if CritTag then
            ExtraResult.ResultTags:Add(CritTag)
            ExtraResult.ResultTags:Add(Crit)-- [NEW]添加暴击Tag用来显示暴击特殊飘字
        end
    end
```
再次启动调试，普通攻击的飘字效果为默认飘字属性配置，当触发暴击伤害时，飘字效果是“DamageNumber.Crit”标签对应的属性配置。

![暴击 .gif](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/FW1Kc%E6%9A%B4%E5%87%BB%20.gif)

- 若伤害目标与造成伤害的玩家不同，则造成伤害的玩家会看到伤害飘字
- 若伤害目标与造成伤害的玩家相同（自己对自己造成的伤害），则除自身外，其他玩家均会看到伤害飘字
