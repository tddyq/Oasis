# 经典海岛模板——Template_Ty_Forest

经典海岛模板用和平海岛地图、包含了跳伞、信号圈、空投、空袭、物资刷新、载具刷新、结算等和平精英经典模式全部流程。
如果你需要基于经典模式做玩法（例如暗夜危机、特种作战等），可基于此创建自己的工程。

<br>

## 模式介绍

相信各位开发者都很熟悉和平精英的经典模式

<br>

### 物资刷新
大地图物资刷新和我们提供的物资刷新模板（Template_SpawnItem）实现方式有所不同，因大地图布物资点工作量巨大，因此如果开发者要删掉和平本来的某种物资，或者新增自己的物资，可采用修改和平的物资投放和概率的方式来进行物资刷新（即本文介绍的方式）；如果需要在大地图的基础上增加少量额外的物资刷新或者小地图的物资刷新，可采用我们提供的物资刷新模板的方式来刷物资

#### 基本原理
场景和建筑中会去布一些刷新点，如图所示(和平已布)：<br>
![企业微信截图_16869054464105.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16869054464105.png)

这些刷新点有两个标签：ESpot Group X 和 ESpot Type X  (其中X代表某个字母），对于所有大地图：
- AA对应的是城区点
- CA是对应的是野外点
- BA、BB对应的是其他点

随后我们只需要配置每种类型的点是否要刷物资（如果刷，刷物资的概率是多少），每次刷几份物资，刷哪几种物资即可

#### 配置方式

在UGCGameMode中，找到Data Manager，对**GMData Source Global Settings**做如下配置。
其中道具生成类中，选择ItemSpawn_Forest，路径位置应选/Game/BluePrints/Core/Forest/Sub/ItemSpawn Forest
如图所示：<br>
![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/YjQQhimage.png)<br>
![企业微信截图_16869054965669.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16869054965669.png)

分别是道具生成器，和道具表，前者约定是否要刷物资，每次刷几份物资，后者决定刷哪几种物资

#### 道具生成器

打开道具生成器（如果开发者想自定义，则基于“海岛道具生成器”继承蓝图进行修改，并修改UGCGameMode中的引用）。<br>

- Item Per Spot Min和Item Per Spot Max:会取min到max中间的一个值（闭区间），去决定这个点刷几次物资<br>

#### 激活游戏状态

<details open>
<summary> <font face="楷体">激活游戏状态</font> </summary>
查看【激活游戏状态】流程，其中创建了一个新的动作（Action），动作类型（Action Type）为执行脚本，脚本名：Action_GameActiveCheck，脚本内容如下：<br>

```lua
--[[------------------------------------------激活游戏状态控制------------------------------------------------------]]--

local Action_GameActiveCheck = {}

function Action_GameActiveCheck:Execute()
	print("Action_GameActiveCheck:Execute " ..tostring(self))
	
	UGCBlueprintFunctionLibrary.InitGameModeStates();
	UGCGameSystem.GameState.GameModeState = "FightingState"
	
	return true;
end

return Action_GameActiveCheck;
```

获取所有玩家的PlayerState,统计现有存活的队伍数量，脚本内容参考如下：

```
local AliveTeamIDTab = {}
    local AllPlayerStates = TeamModeComponent:GetTeamates(-1)
    for i, PlayerStateWeakPtr in ipairs(AllPlayerStates) do
        local PlayerState = PlayerStateWeakPtr:Get()
        
        if PlayerState then
            print("Action_GameEndCheck:Execute AllPlayer PlayerName = "..PlayerState.PlayerName.." TeamID = "..PlayerState.TeamID)
        end

        if PlayerState and PlayerState:IsAlive() then       
            local bCanInsert = true
            local NewTeam = PlayerState.TeamID
            for index, value in ipairs(AliveTeamIDTab) do
                if value == NewTeam then
                    bCanInsert = false
                end
            end
            if bCanInsert then
                print("Action_GameEndCheck:Execute NewTeam = "..NewTeam)
                table.insert(AliveTeamIDTab, NewTeam)
            end
        end
    end
```

只有一个队伍存活时，游戏结束。给第一名和第二名赋值名次，然后通知所有人游戏结束事件UGCGameEventType.ShowSettlement，显示结算信息。最后调用GameEnd事件发送游戏结束协议，关闭ds。

当剩余队伍数量大于1时，游戏未结束。此时统计当前队伍是否团灭：1.团灭：计算当前队伍排名并通知所有队员显示结算信息。2未团灭：计算死亡玩家的排名并通知显示结算信息。

```
local TeamatesPlayerStates = {}
        local VictimState = ScriptGameplayStatics.GetPlayerStateByPlayerKey(self, self.VictimKey)
        if VictimState then
            TeamatesPlayerStates =TeamModeComponent:GetTeamates(VictimState.TeamID)  
            print("Action_GameEndCheck:Execute Victim PlayerName = "..VictimState.PlayerName.." TeamID = "..VictimState.TeamID.." #TeamatesPlayerStates = "..#TeamatesPlayerStates)
        end

        --检查队友存活数量
        local bTeamatesAllDead = true
        for i, PlayerStateWeakPtr in ipairs(TeamatesPlayerStates) do
            local PlayerState = PlayerStateWeakPtr:Get()
            if PlayerState and PlayerState:IsAlive() then
                bTeamatesAllDead = false
            end
        end
    
        if bTeamatesAllDead then

            local GameResultData = {}
            GameResultData.bGameEnd = false
            GameResultData.bShowTeamatesInfo = true
            GameResultData.TeamRank = #AliveTeamIDTab + 1

            for i, PlayerStateWeakPtr in ipairs(TeamatesPlayerStates) do
                local PlayerState = PlayerStateWeakPtr:Get()       
                if PlayerState then
                    PlayerState.TeamRank = #AliveTeamIDTab + 1
                    print("Action_GameEndCheck:Execute Teamates PlayerName = "..PlayerState.PlayerName.." PlayerState.TeamID = "..PlayerState.TeamID)
    
                    local PlayerController = ScriptGameplayStatics.GetPlayerControllerByPlayerKey(self, PlayerState.PlayerKey);
                    if PlayerController then
                        UnrealNetwork.CallUnrealRPC(PlayerController, PlayerController, "ClientRPC_SendEvent", UGCGameEventType.ShowSettlement, GameResultData)
                    end
                end
            end
        else
            --检查玩家数量
            local AliveNum = UGCGameSystem.GameState:GetAliveRealPlayerNum()
            VictimState.TeamRank = AliveNum + 1
            print("Action_GameEndCheck:Execute VictimState.TeamRank = "..VictimState.TeamRank);
    
            local GameResultData = {}
            GameResultData.bGameEnd = false
            GameResultData.bShowTeamatesInfo = false
            GameResultData.TeamRank = AliveNum + 1

            local VictimController = ScriptGameplayStatics.GetPlayerControllerByPlayerKey(self, VictimState.PlayerKey);
            if VictimController then
                UnrealNetwork.CallUnrealRPC(VictimController, VictimController, "ClientRPC_SendEvent", UGCGameEventType.ShowSettlement, GameResultData)
            end
        end
```
