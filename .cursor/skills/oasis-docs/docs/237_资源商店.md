# 资源商店

资源商店提供了包含 [商城模板](https://developer.gp.qq.com/wikieditor/#/catalog/20026)、[抽奖模板](https://developer.gp.qq.com/wikieditor/#/catalog/20027) 等各类功能模板及功能蓝图资源，开发者下载并部署到工程后即可使用这些资源的功能。

<br>

## 资源商店界面

编辑器菜单栏点击【资源商店】按钮以呼出资源商店界面。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/RviLpimage.png)

资源商店界面包含公共资源、官方资源、我的下载、我的收藏及我的上传五个页签，在页签下方提供了按指定条件检索的搜索功能，右侧区域为资源的详情栏。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/TikXKimage.png)

- 公共资源：所有共享的资源，包含绿洲启元官方资源和开发者上传资源
- 官方资源：绿洲启元官方提供的资源
- 我的下载：已下载的资源
- 我的收藏：已收藏的资源
- 我的上传：开发者本地上传的资源，当审核通过后将发布至云端并共享下载

> 当前暂未开放上传资源功能

选中任意一个资源，将在右侧详情栏展示该资源的详细信息，包括资源的简介、资源的发布及更新日期、资源大小等等。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/BZsYPimage.png)

<br>

## 资源下载

在资源商店界面查找并选中需要下载的目标资源，在资源详情栏中点击【下载】按钮。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/VZQNYimage.png)

下载后的资源默认不会直接加载到工程内，而是暂存到公共目录中，此时该资源对其他工程也可见，各工程无需重复下载即可从公共目录将该资源按需导入工程。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/0ZGaFimage.png)

> 公共目录路径：``rail_apps/OasisEraEditor(2001776)/ShadowTrackerExtra/Saved/UGCExtendsCache``

<br>

## 导入及使用资源

从公共目录点击需要使用的目标资源，弹出是否需要导入的确认框，点击【是】则将该资源导入并加载到工程内，存放路径统一为 ``ExtendResource`` 目录下。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/tj7vDimage.png)

资源导入后可按照规定的要求或规则使用，各类资源的使用方法可参考对应的功能文档。

<br>

## 收藏资源

如果希望将资源加入收藏夹以方便查找，可在资源的详情栏中点击【收藏】按钮。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/TMU0Nimage.png)

成功收藏的资源将出现在 ``我的收藏`` 页签下。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/nAz7Gimage.png)

再次点击【收藏】按钮将取消收藏，并从 ``我的收藏`` 页签下被移除。
