## 1.35.12.27844 更新说明

## 1.35.12.27840 版本说明

### 编辑器

**新功能**

- 新增 [场景破坏物](https://developer.gp.qq.com/wikieditor/#/catalog/20293) 蓝图模板，适用于具备生命值属性及行为特性的静态物件对象
