## 1.29.13.13015 版本说明

- ActorComponent子组件支持属性复制与RPC

#### bug Fixes

- 修复主角Pawn缩放后站、蹲、趴等状态异常问题
- 修复地表画刷工具偶现贴图异常问题
- 修复某些条件下玩家骑乘马载具出现APose问题
- 修复海岛地图偶现无法加载建筑物的问题
- 修复二次匹配功能在子模式中获取GameModeID为0的问题
- 修复二次匹配功能极端情况下存档数据脏读问题
- 修复多航线场景下飞行参数设置不生效问题
- 修复商业化购买功能失效问题
- 修复躲猫猫模板变身失效问题
- 修复多种保存场景下触发编辑器崩溃问题
- 修复坦克蓝图炮台旋转偏移问题
