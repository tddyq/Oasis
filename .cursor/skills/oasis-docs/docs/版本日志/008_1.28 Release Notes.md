## 1.28.13 更新说明

### 编辑器

#### 功能更新

- 新增 [经典团竞](https://developer.gp.qq.com/wikieditor/#/catalog/20015) 和 [军备团竞](https://developer.gp.qq.com/wikieditor/#/catalog/20016) 两个案例模板
- 新增 [商城模板](https://developer.gp.qq.com/wikieditor/?timeStamp=1725589224096#/catalog/20026) 与 [抽奖模板](https://developer.gp.qq.com/wikieditor/?timeStamp=1725589224096#/catalog/20027)，助力开发者快速构建商城与抽奖系统，丰富玩法商业化功能
- 全新 [天气系统控件](https://developer.gp.qq.com/wikieditor/#/catalog/225)，实现更精细的光照调控与动态天气效果
- 支持怪物之间的 [动画蓝图复用](https://developer.gp.qq.com/wikieditor/#/catalog/20017)，不同骨骼的怪物都能正确播放姿态动画
- [武装载具](https://developer.gp.qq.com/wikieditor/#/catalog/20025) 已支持动态切换武器，以及设置指定武器的可用性

#### bug Fixes

- 修复更换角色Mesh概率出现崩溃的问题
- 修复Lua资源引用工具的空面板问题
- 修复骑马动画在部分场景下出现APose的问题
- 修复海岛大地图PIE调试下物资刷新点位异常及移动端不刷新物资问题
- 修复多Predict组件触发崩溃问题
- 修复接入二次匹配功能的玩法无法进入默认子模式的问题

<br>

### 管理平台

#### 功能更新

- 开发者可上报玩法灰度期间的 [自定义行为数据](https://developer.gp.qq.com/wikieditor/#/catalog/20023)，管理平台支持上报数据的统计提取
- 优化开启测试的添加白名单功能，单玩法下可统一管理白名单，不用再区分版本
- 新增玩法信息更新生效状态，方便开发者了解信息更新的生效情况
- 新增玩家反馈预警消息通知，帮助开发者快速获取玩法的主要bug反馈
- 新增签约消息提醒，帮助开发者及时进行相关签约
- 新增抢先体验上下架状态展示，帮助开发者了解玩法在抢先体验的上下架状态
- 异常上报和项目日志打通团队查看权限，团队内上传玩法的异常上报、项目日志都可以查看了
- 优化运营数据页面展示，帮助更高效查看相关运营数据
- 供应商入驻流程和玩法解绑，可以直接通过账号管理页面进行供应商入驻

<br>

### API

#### 功能更新

- 新增进入/退出跳伞状态的函数 [``TryEnterParachuteState``](https://developer.gp.qq.com/api/#/searchContent/UGCPawnSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2F%E8%A7%92%E8%89%B2%2FUGCPawnSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCPawnSystem) 与 [``ExitParachuteState``](https://developer.gp.qq.com/api/#/searchContent/UGCPawnSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2F%E8%A7%92%E8%89%B2%2FUGCPawnSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCPawnSystem)
- [``HiddenBoneByBoneName``](https://developer.gp.qq.com/api/#/searchContent/UGCPawnSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2F%E8%A7%92%E8%89%B2%2FUGCPawnSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCPawnSystem) 支持动态隐藏玩家Pawn的指定骨骼

#### bug Fixes

- 修复 ``SetReloadTime`` 和 ``SetTacticalReloadTime`` 在枪械装配快速弹匣或快速扩容弹匣时，换弹时间异常问题
- 修复枪械调用 ``StopFire`` 概率性无法停止开火的问题
