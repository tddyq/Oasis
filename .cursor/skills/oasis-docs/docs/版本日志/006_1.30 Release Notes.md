## 1.30.21.13232 版本说明

### 编辑器

**功能更新**

- 新增 [阶段匹配](https://developer.gp.qq.com/wikieditor/#/catalog/20062) 功能，允许动态降低匹配人数条件，提高匹配成局的成功率
- 支持开发者通过 [通用掉落](https://developer.gp.qq.com/wikieditor/#/catalog/20058) 的配置实现随机产出物的功能效果
- [数据存档](https://developer.gp.qq.com/wikieditor/#/catalog/20059?autoJump=%E5%AD%98%E6%A1%A3%E8%B0%83%E8%AF%95) 功能已支持Debug环境下（PIE）调试，支持一次调试会话内的跨DS存储
- [二次匹配](https://developer.gp.qq.com/wikieditor/#/catalog/20061?autoJump=%E5%A4%9A%E6%A8%A1%E5%BC%8F%E8%B0%83%E8%AF%95) 功能已支持Debug环境下（PIE）调试
- 新增⌜实验性⌟功能：[新技能编辑器](https://developer.gp.qq.com/wikieditor/#/catalog/20073)

**bug Fixes**

- 收到开发者们的反馈建议，对商城模板、抽奖模板和排行榜模板进行了重构优化和bug修复，将逐步通过资源商店更新各模板
- 修复C4炸弹在PIE调试时出现的UI缺失问题
- 修复当导航网格文件navmesh太大时，LoadMap加载异常导致移动端无法正常进入游戏的问题

<br>

### 管理平台

**功能更新**

- 玩法上架的消息提醒与邮件通知，调整为只对玩法首次上架生效
- 玩家反馈功能支持筛选带图片/视频的内容，方便开发者筛选更有效的反馈信息
- 运营数据页面新增玩法排名趋势图，开发者可以更直观地了解玩法表现趋势
- 玩法提审页面移除“最小内存限制”、“安全检测”等填写项

<br>

### API

**功能更新**

- [``SetSprintSpeedScale``](https://developer.gp.qq.com/api/#/searchContent/UGCPawnAttrSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2F%E8%A7%92%E8%89%B2%2FUGCPawnAttrSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCPawnAttrSystem&autoJump=SetSprintSpeedScale) 调整为仅对疾跑速度生效，如果需要变更玩家普通移动速度，请使用 [``SetWalkSpeedScale``](https://developer.gp.qq.com/api/#/searchContent/UGCPawnAttrSystem?classDetailShow=true&path=class%2Fdetail%2F%E5%92%8C%E5%B9%B3%E5%85%A8%E5%B1%80%E6%8E%A5%E5%8F%A3%2F%E8%A7%92%E8%89%B2%2FUGCPawnAttrSystem.json&isSelect=1&apiEnc=%5B%22%E7%B1%BB%22%5D&apiLabel=UGCPawnAttrSystem&autoJump=SetWalkSpeedScale)
