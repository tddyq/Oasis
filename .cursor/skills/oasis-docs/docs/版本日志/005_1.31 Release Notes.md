## 1.31.10.31450 版本说明

### 编辑器

**功能更新**

- 全新 [技能编辑器](https://developer.gp.qq.com/wikieditor/#/catalog/20091) 与 [Buff编辑器](https://developer.gp.qq.com/wikieditor/#/catalog/20087) ，支持蓝图形式快速配置多类型技能与Buff，助力开发者构建玩法技能体系
- 全新 [物品编辑器](https://developer.gp.qq.com/wikieditor/#/catalog/20101) 与 [背包系统](https://developer.gp.qq.com/wikieditor/#/catalog/20104)，物品的自定义配置与使用更加灵活
- 新增 [Gameplay标签](https://developer.gp.qq.com/wikieditor/#/catalog/20102) 方便对游戏对象进行分类和标记，可配套应用于技能编辑器、物品编辑器等多种系统功能中
- 新增 [属性管理器](https://developer.gp.qq.com/wikieditor/#/catalog/20098) 提供统一创建及访问游戏对象属性的方式
- 支持通过 [伤害公式](https://developer.gp.qq.com/wikieditor/#/catalog/20099) 功能管理全局伤害计算逻辑
- 支持通过 [UISlot锚点](https://developer.gp.qq.com/wikieditor/#/catalog/20097) 将自定义控件添加至和平主界面，同时 [WidgetLayout](https://developer.gp.qq.com/wikieditor/#/catalog/20096) 已支持可视化的添加及布局方式
- 支持为玩家分配不同的 [阵营](https://developer.gp.qq.com/wikieditor/#/catalog/20095)，拓展玩法竞技对抗深度
- [客户端调试管理器](https://developer.gp.qq.com/wikieditor/#/catalog/20042) 支持模拟 [玩家离开游戏](https://developer.gp.qq.com/wikieditor/#/catalog/157?autoJump=%E7%8E%A9%E5%AE%B6%E7%A6%BB%E5%BC%80%E6%B8%B8%E6%88%8F) 事件
- 优化玩法工程上传速度，显著缩短上传时间
- 优化编辑器Debug调试的启动速度

**bug Fixes**

- 修复资源目录下的音频资源点击过ModifyEvent后，启动PIE调试报错的问题
- 修复部分资源目录下的UI控件模板引用资源丢失问题
- 修复角色使用药品或者平底锅攻击无动画的问题
- 修复工程上传后偶现资源丢失的问题
- 修复C4炸弹无伤害的问题
- 修复被RPG类武器击杀复活后偶现黑屏的问题
- 修复切换工程的情境下无法保存蓝图的问题

<br>
