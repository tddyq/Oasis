## 1.32.11.31531 版本说明

### 编辑器

**新功能**

- 新增 [通用GM面板](https://developer.gp.qq.com/wikieditor/#/catalog/20109) ，预置多种常用GM功能且支持扩展自定义GM
- 新增 [通用掉落组件](https://developer.gp.qq.com/wikieditor/#/catalog/20142)，为游戏对象添加随机掉落功能更加方便
- 新增 [通用运动组件](https://developer.gp.qq.com/wikieditor/#/catalog/20150)，实现万物皆可动
- 新增 [通用伤害飘字](https://developer.gp.qq.com/wikieditor/#/catalog/20160)，玩家角色与怪物均可接入伤害飘字效果
- 新增 [通用消息系统](https://developer.gp.qq.com/wikieditor/#/catalog/20163)，支持对特定对象或者全局广播消息
- 新增 [通用抛体](https://developer.gp.qq.com/wikieditor/#/catalog/20165)，可自定义飞行轨迹、命中过滤器及命中后的执行动作，实现更复杂的抛体功能
- 全新怪物系统，包含 [新的怪物基类](https://developer.gp.qq.com/wikieditor/#/catalog/20144)、行为树编辑器、[怪物血条](https://developer.gp.qq.com/wikieditor/#/catalog/20145)、[怪物逻辑组件](https://developer.gp.qq.com/wikieditor/#/catalog/20162)、[怪物刷新器](https://developer.gp.qq.com/wikieditor/#/catalog/20156) 及 [怪物调试工具](https://developer.gp.qq.com/wikieditor/#/catalog/20161) 等多种新功能，并且新怪物已全面支持技能和Buff编辑器，为开发者创造个性化怪物提供更多可能性

**功能更新**

【商业化及功能模板】

- 各功能模板已全部接入 [GamePart模块化](https://developer.gp.qq.com/wikieditor/#/catalog/20090) 配置方式，通过 [虚拟物品](https://developer.gp.qq.com/wikieditor/#/catalog/20139) 管理物品数据的流转变化

【技能编辑器】

- 技能Event新增 [累计受到伤害事件](https://developer.gp.qq.com/wikieditor/#/catalog/20112?autoJump=%E7%B4%AF%E8%AE%A1%E5%8F%97%E5%88%B0%E4%BC%A4%E5%AE%B3%E4%BA%8B%E4%BB%B6)、[累计造成伤害事件](https://developer.gp.qq.com/wikieditor/#/catalog/20112?autoJump=%E7%B4%AF%E8%AE%A1%E9%80%A0%E6%88%90%E4%BC%A4%E5%AE%B3%E4%BA%8B%E4%BB%B6)、[背包事件](https://developer.gp.qq.com/wikieditor/#/catalog/20112?autoJump=%E8%83%8C%E5%8C%85%E4%BA%8B%E4%BB%B6)、[角色落地事件](https://developer.gp.qq.com/wikieditor/#/catalog/20112?autoJump=%E8%A7%92%E8%89%B2%E8%90%BD%E5%9C%B0%E4%BA%8B%E4%BB%B6) 和 [技能事件](https://developer.gp.qq.com/wikieditor/#/catalog/20112?autoJump=%E6%8A%80%E8%83%BD%E4%BA%8B%E4%BB%B6) 等多种事件组件
- 技能Condition新增 [背包物品条件](https://developer.gp.qq.com/wikieditor/#/catalog/20111?autoJump=%E8%83%8C%E5%8C%85%E7%89%A9%E5%93%81%E6%9D%A1%E4%BB%B6) 和 [队伍存活人数](https://developer.gp.qq.com/wikieditor/#/catalog/20111?autoJump=%E9%98%9F%E4%BC%8D%E5%AD%98%E6%B4%BB%E4%BA%BA%E6%95%B0) 条件组件，增加通用逻辑运算符配置
- 技能Task新增 [基础动画](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E5%9F%BA%E7%A1%80%E5%8A%A8%E7%94%BBTask)、[技能动画](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BD%E5%8A%A8%E7%94%BBTask)、[追踪目标](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E8%BF%BD%E8%B8%AA%E7%9B%AE%E6%A0%87)、[生成Monster](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E7%94%9F%E6%88%90Monster)、[背包操作](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E8%83%8C%E5%8C%85%E6%93%8D%E4%BD%9C)、[近战攻击](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E8%BF%91%E6%88%98%E6%94%BB%E5%87%BB)、[武器属性修改](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E6%AD%A6%E5%99%A8%E5%B1%9E%E6%80%A7%E4%BF%AE%E6%94%B9)、[下一次伤害属性修改](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E4%B8%8B%E4%B8%80%E6%AC%A1%E4%BC%A4%E5%AE%B3%E5%B1%9E%E6%80%A7%E4%BF%AE%E6%94%B9)、[显示UI](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E6%98%BE%E7%A4%BAUI) 与 [进度条UI](https://developer.gp.qq.com/wikieditor/#/catalog/20094?autoJump=%E6%8A%80%E8%83%BDTask-%E8%BF%9B%E5%BA%A6%E6%9D%A1UI) 等Task组件
- 新增 [技能状态图](https://developer.gp.qq.com/wikieditor/#/catalog/20152)，支持可视化的方式编辑技能阶段及跳转条件
- 新增 [技能调试工具](https://developer.gp.qq.com/wikieditor/#/catalog/20114)，支持运行时调试技能、Buff与属性
- [序列播放器](https://developer.gp.qq.com/wikieditor/#/catalog/20091?autoJump=%E5%BA%8F%E5%88%97%E6%92%AD%E6%94%BE%E5%99%A8%E9%85%8D%E7%BD%AE) 新增技能动画轨道，轨道支持设置激活条件
- 被动技能支持 [合并条件](https://developer.gp.qq.com/wikieditor/#/catalog/20171?autoJump=ECA%E9%85%8D%E7%BD%AE) 配置
- 技能与Buff中的属性修改实现调整为基于 [属性修改器](https://developer.gp.qq.com/wikieditor/#/catalog/20153) 的计算框架，支持临时与永久属性修改
- 技能基础配置中的消耗属性方式调整为固定的减法运算

【物品编辑器】

- 新增近战武器、投掷物、穿戴装备、配件和子弹物品模板，覆盖和平精英各类常用武器
- 新增 [物资刷新器](https://developer.gp.qq.com/wikieditor/#/catalog/20155)，支持蓝图配置的方式管理物资刷新流程与规则
- 新增 [近战武器](https://developer.gp.qq.com/wikieditor/#/catalog/20158) 与 [投掷物](https://developer.gp.qq.com/wikieditor/#/catalog/20157) 实体蓝图模板，支持自定义近战武器与投掷物
- 枪械物品与装备物品支持通过 [槽位配置](https://developer.gp.qq.com/wikieditor/#/catalog/20101?autoJump=V2%E8%83%8C%E5%8C%85%E5%B1%9E%E6%80%A7%E9%85%8D%E7%BD%AE) 添加自定义配件

【背包系统】

- 优化背包组件与背包UI组件的创建方式 

【角色系统】

- 新增 [死亡掉落](https://developer.gp.qq.com/wikieditor/#/catalog/20148) 配置，支持自定义玩家角色的物品掉落与保留规则
- [阵营系统](https://developer.gp.qq.com/wikieditor/#/catalog/20095?autoJump=%E9%98%B5%E8%90%A5%E7%B3%BB%E7%BB%9F) 支持为新怪物设置阵营，调整玩家角色阵营的默认分配规则与设置方式

**bug Fixes**

- 修复外研线客户端无法下载和平大地图资源的问题
- 修复移动端预设技能UI槽位置异常的问题
- 修复编辑场景内带有序列播放器的蓝图实例对象时，触发编辑器崩溃的问题
- 修复删除 ``DA_GameModeGeneral`` 数据资产蓝图后无法再次创建的问题
- 修复部分属性中文搜索失效的问题
- 修复抛物线方向选择器可见性属性设置不生效的问题
- 修复widgetlayout无法隐藏队友状态栏的问题
- 修复商城模板不显示绿洲币状态栏及弱网重连后无法购买商品的问题
- 修复电击步枪部分子弹无伤害的问题
- 修复被动技能中添加冲量Task无效的问题
- 修复被动技能中选择方向与选择目标Task的指示器不销毁的问题
