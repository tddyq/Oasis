# 3D坐标转换屏幕2D坐标

- 目标Location在移出目标后不会显示，Success将为false
``` lua
local Location = {0,0,0} --Location为需要转换的3D坐标
local SizeX,SizeY = PlayerController:GetViewportSize()
local Scale = WidgetLayoutLibrary.GetViewportScale(GameFrontendHUD)
local Success,ScreenPos = PlayerController:ProjectWorldLocationToScreen(Location)
ScreenPos.x = (ScreenPos.x - SizeX/2) / Scale
ScreenPos.y = (ScreenPos.y - SizeY/2) / Scale
```


- 目标Location移出屏幕后返回边缘的ScreenPos

``` lua
local Limit = {X=10, Y=100, Z=10, W=150}
local ScreenPos, Angle = STExtraBlueprintFunctionLibrary.GetScreenLocationOfWorldLoc(WorldContextObject,Location,Limit)
```
- Limit参数是指和边界的间隔
