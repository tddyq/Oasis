## MapMarkWidget相关设置
### Widget锚点坐标设置
1. Widget结构为CanvasPanel下放置所需显示控件
2. 控件的锚点设置为中心（0.5,0.5），Alignment设置为中心（0.5,0.5）
3. 控件的位置设置为中心（0,0）
4. 控件的大小推荐在30左右，非必须，仅为推荐值

### PlayerMark相关配置参数（仅用于需要跟随目标移动的PlayerMark）

蓝图页面右上角点击图标进入数据配置界面。

![002.4d7a35c1[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/002.4d7a35c1%5B1%5D.png)

勾选DynamicPos和RotateWidgetToAngle参数，分别对应动态更新

![003.c7bae612[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/003.c7bae612%5B1%5D.png)

UsePlayerState参数：false数据来源为PlayerPawn的前提下，如果仅需要同步队友那么推荐勾选，如果同步对象为敌人则不勾选，如果不勾选的情况下，PlayerPawn超出了网络同步距离被裁剪的时候，小地图标点也会消失，如果勾选则会持续同步，但是仅支持同步给队友。
