## 添加切换颜色的代码

- 给Actor的Box添加仅客户端触发的代码，当角色进入Box范围内，触发修改材质颜色

``` lua

---@field DefaultSceneRoot USceneComponent

--Edit Below--

local TestCube = {};

TestCube.DynamicMaterial = nil



function TestCube:ReceiveBeginPlay()

    self.SuperClass.ReceiveBeginPlay(self);

    

    if not UGCGameSystem.IsServer() then

        --创建材质实例

        local MaterialInst = UE.LoadObject(UGCGameSystem.GetUGCResourcesFullPath('Asset/Blueprint/TestCubeMaterial.TestCubeMaterial'))

        self.DynamicMaterial = self.Cube:CreateAndSetMaterialInstanceDynamicFromMaterial(0, MaterialInst)



        self.Box.OnComponentBeginOverlap:Add(self.Box_OnComponentBeginOverlap,self)

    end

end



function TestCube:Box_OnComponentBeginOverlap(OverlappedComponent, OtherActor, OtherComp, OtherBodyIndex, bFromSweep, SweepResult)

    ugcprint("TestCube:Box_OnComponentBeginOverlap")

    --替换颜色为黄色

    self.DynamicMaterial:SetVectorParameterValue("Color_Modifier",{R = 1, G = 1, B = 0, A = 0})

end

```

## 其他注意事项

- 此教学代码仅演示在客户端运行的情况，没有处理断线重连或者杀进程重进游戏的状态恢复，如果需要处理，需要自行服务器记录颜色状态，并处理弱网和重进逻辑恢复材质颜色
