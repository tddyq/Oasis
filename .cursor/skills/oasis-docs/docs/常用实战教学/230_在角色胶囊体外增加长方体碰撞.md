### 新增BOX并添加tag的方法：

- 打开UGCPlayerPawn的蓝图

![001.a21d62bd[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/001.a21d62bd%5B1%5D.png)

![002.12936e63[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/002.12936e63%5B1%5D.png)

- BOX添加Tag IgnoreComponentSetDeactivate
