# 绿洲启元编辑器文档查询

查询和平精英绿洲编辑器官方Wiki文档。

**文档根路径**：本 skill 目录下的 `docs/` 子目录。作为项目级 Skill 安装到当前仓库的 `.cursor/skills/oasis-docs/` 后，完整路径为 `.cursor/skills/oasis-docs/docs/`。

## 查询流程

1. 根据用户问题，从下方索引定位相关文件编号和分类目录
2. 如果索引中能明确找到对应文件，直接用 Read 工具读取该文件。注意：必须拼接文档根路径 + 分类目录名，例如 `.cursor/skills/oasis-docs/docs/进阶内容/168_信号圈系统.md`
3. 如果不确定哪个文件包含答案，使用搜索工具在对应分类目录下搜索中文关键词
4. 如果需要跨分类搜索，在整个 docs 目录下搜索
5. 根据搜索结果，用 Read 读取最相关的文件（指定 offset/limit 读取关键段落）
6. 用中文总结回答，引用具体文件名方便用户回查

## 注意事项

- 用户主要使用可视化编辑器操作，很少写Lua代码。回答时优先说明编辑器内的操作步骤
- 用户主要做PVE玩法（打怪闯关），怪物、技能、Buff、物资、关卡相关优先
- 每次只读取需要的文件，不要一次性读取多个无关文件
- 文件名格式为 `编号_标题.md`，编号可用于快速定位
